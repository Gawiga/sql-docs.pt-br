# <a name="proxy-account-properties---new-proxy-account-principals-tab"></a>Propriedades da conta proxy – Nova conta proxy (guia Entidades)
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] Use essa página para exibir ou alterar os principais que podem usar uma conta proxy do [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)] Agent em etapas de trabalho.  
  
## <a name="options"></a>Opções  
**Principais da conta proxy**  
Lista os principais que podem usar essa conta proxy.  
  
**Adicionar**  
Adiciona um principal à lista.  
  
**Remover**  
Remove o principal selecionado da lista.

# <a name="autosize-selected-tables-in-diagrams-visual-database-tools"></a>Dimensionar automaticamente tabelas selecionadas em diagramas (Visual Database Tools)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]
Você pode rapidamente encaixar tabelas em seu diagrama para o menor tamanho que exibir todos os campos.  
  
### <a name="to-automatically-size-selected-tables-in-diagrams"></a>Para dimensionar automaticamente as tabelas selecionadas em diagramas  
  
1.  Selecione a(s) tabela(s) que você quer redimensionar.  
  
2.  Clique com o botão direito do mouse em uma tabela selecionada e, no menu de atalho, escolha **Autodimensionar Tabelas Selecionadas**.

# <a name="modify-queries-visual-database-tools"></a>Modificar consultas (Visual Database Tools)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)] Abra uma consulta para modificá-la no Designer de Exibição e Consulta.  
  
### <a name="to-open-a-query"></a>Para abrir uma consulta  
  
1.  No menu **Exibir** , clique em **Gerenciador de Soluções**.  
  
2.  No Gerenciador de Soluções, clique com o botão direito do mouse na consulta que desejar abrir e clique em **Abrir**.  
  
3.  Para modificar a consulta no Designer de Consultas, realce as instruções SQL, clique com o botão direito do mouse na área realçada e selecione **Projetar Consulta no Editor**.  
  
O Designer de Consulta está agora aberto e você pode fazer modificações.

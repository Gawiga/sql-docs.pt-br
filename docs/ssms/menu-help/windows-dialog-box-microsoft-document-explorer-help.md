# <a name="windows-dialog-box-microsoft-document-explorer-help"></a>Caixa de diálogo Windows (Ajuda do Gerenciador de Documentos da Microsoft)
Essa caixa de diálogo permite exibir, fechar e organizar documentos que estão abertos em editores e designers. Você pode exibir essa caixa de diálogo clicando em **Windows** no menu **Janela** .  
  
## <a name="options"></a>Opções  
**Nome**  
Exibe os nomes de documento e janelas de ferramenta, com o item ativo atualmente selecionado.  
  
**Caminho**  
Exibe o caminho completo para documentos. Normalmente, o Windows não tem um caminho.  
  
**Ativar**  
Torna a janela do documento ou de ferramentas selecionada o item ativo no Gerenciador de Documentos do [!INCLUDE[msCoName](../../includes/msconame_md.md)] .  
  
**Salvar**  
Salva a janela selecionada. Nem todas as janelas podem ser salvas.  
  
**Fechar o Windows**  
Fecha os documentos e janelas selecionados na lista.

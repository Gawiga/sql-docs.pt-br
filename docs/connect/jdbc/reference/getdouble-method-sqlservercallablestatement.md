# <a name="getdouble-method-sqlservercallablestatement"></a>Método getDouble (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera o valor do parâmetro designado como um **duplo** na linguagem de programação Java.  
  
## <a name="overload-list"></a>Lista de sobrecargas  
  
|Nome|Description|  
|----------|-----------------|  
|[getDouble (int)](../../../connect/jdbc/reference/getdouble-method-int.md)|Recupera o valor do parâmetro designado como um **duplo** no considerando o índice do parâmetro de linguagem de programação Java.|  
|[getDouble (Java)](../../../connect/jdbc/reference/getdouble-method-java-lang-string.md)|Recupera o valor do parâmetro designado como um **duplo** no considerando o nome do parâmetro de linguagem de programação Java.|

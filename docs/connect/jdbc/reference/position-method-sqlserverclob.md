# <a name="position-method-sqlserverclob"></a>Método position (SQLServerClob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Retorna a posição do caractere do objeto CLOB especificado ou da subcadeia de caracteres no CLOB com base na posição inicial fornecida.

|----------|-----------------|  
|[posição (CLOB, long)](../../../connect/jdbc/reference/position-method-java-sql-clob-long.md)|Retorna a posição de caractere do objeto CLOB especificado no CLOB com base na posição inicial fornecida.|  
|[posição (Java, long)](../../../connect/jdbc/reference/position-method-java-lang-string-long.md)|Retorna a posição de caractere da subcadeia de caracteres especificada no CLOB com base na posição inicial fornecida.|

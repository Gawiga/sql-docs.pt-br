# <a name="getdatetimeoffset-method-sqlservercallablestatement"></a>Método getDateTimeOffset (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Esse método foi adicionado no [!INCLUDE[msCoName](../../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion_md.md)] JDBC Driver 3.0.  
  
 Recupera o valor do parâmetro designado como um [DateTimeOffset classe](../../../connect/jdbc/reference/datetimeoffset-class.md) objeto considerando o índice do parâmetro de linguagem de programação Java.  
  
 Você pode definir um [DateTimeOffset classe](../../../connect/jdbc/reference/datetimeoffset-class.md) valor de parâmetro com [Setdatetimeoffset](../../../connect/jdbc/reference/setdatetimeoffset-method-sqlservercallablestatement.md).  
  
## <a name="overload-list"></a>Lista de sobrecargas  
  
|Nome|Description|  
|----------|-----------------|  
|[getDateTimeOffset(int)](../../../connect/jdbc/reference/getdatetimeoffset-method-int.md)|Recupera o valor do parâmetro designado como um [DateTimeOffset classe](../../../connect/jdbc/reference/datetimeoffset-class.md) objeto considerando o índice do parâmetro de linguagem de programação Java.|  
|[getDateTimeOffset(String)](../../../connect/jdbc/reference/getdatetimeoffset-method-string.md)|Recupera o valor do parâmetro designado como um [DateTimeOffset classe](../../../connect/jdbc/reference/datetimeoffset-class.md) objeto considerando o índice do parâmetro de linguagem de programação Java.|

# <a name="sqlserversavepoint-constructor-sqlserverconnection-javalangstringname"></a>Construtor SQLServerSavepoint (SQLServerConnection, java.lang.StringName)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Inicializa uma nova instância do [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md) classe baseia o nome e uma determinada conexão.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
public SQLServerSavepoint(SQLServerConnection con,  
                          java.lang.String sName)  
```  
  
#### <a name="parameters"></a>Parâmetros  
 *Con*  
  
 Um [SQLServerConnection](../../../connect/jdbc/reference/sqlserverconnection-class.md) objeto.  
  
 *sName*  
  
 Um **cadeia de caracteres** que contém o nome do ponto de salvamento.

# <a name="getobject-method-sqlservercallablestatement"></a>Método getObject (SQLServerCallableStatement)

|----------|-----------------|

|[getObject (Java, java.util.Map)](../../../connect/jdbc/reference/getobject-method-java-lang-string-java-util-map.md)|Recupera o valor do parâmetro designado como um objeto na linguagem de programação Java considerando o nome do parâmetro, usando o objeto de mapa fornecido.|

# <a name="getint-method-sqlserverresultset"></a>getInt método (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera o valor da coluna designada na linha atual deste [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) de objeto como um **int** na linguagem de programação Java.

|----------|-----------------|  
|[getInt (int)](../../../connect/jdbc/reference/getint-method-int-sqlserverresultset.md)|Recupera o valor do índice de coluna designada na linha atual deste [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) de objeto como um **int** na linguagem de programação Java.|  
|[getInt (Java)](../../../connect/jdbc/reference/getint-method-java-lang-string-sqlserverresultset.md)|Recupera o valor do nome da coluna designada na linha atual deste [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) de objeto como um **int** na linguagem de programação Java.|

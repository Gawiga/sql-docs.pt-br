# <a name="setapplicationintent-method-sqlserverdatasource"></a>Método setApplicationIntent (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Define o valor da **applicationIntent** propriedade de conexão.  
  
## <a name="syntax"></a>Sintaxe  
  
```vb  
public void setApplicationIntent(java.lang.String applicationIntent);  
```  
  
#### <a name="parameters"></a>Parâmetros  
 *applicationIntent*  
  
 O novo valor da **applicationIntent** propriedade de conexão.  
  
## <a name="remarks"></a>Comentários  
 Para obter mais informações sobre o **applicationIntent** propriedade de conexão, consulte [definindo as propriedades de Conexão](../../../connect/jdbc/setting-the-connection-properties.md).

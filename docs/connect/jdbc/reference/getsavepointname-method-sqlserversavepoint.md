# <a name="getsavepointname-method-sqlserversavepoint"></a>Método getSavepointName (SQLServerSavepoint)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Obtém o nome do ponto de salvamento.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
public java.lang.String getSavepointName()  
```  
  
## <a name="return-value"></a>Valor de retorno  
 Um **cadeia de caracteres** que contém o nome do ponto de salvamento.  
  
## <a name="exceptions"></a>Exceções  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Comentários  
 Esse método getSavepointName é especificado pelo método getSavepointName na interface Java.SQL. savepoint.

# <a name="getprocedureterm-method-sqlserverdatabasemetadata"></a>Método getProcedureTerm (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera o termo preferencial para "procedimento" no banco de dados em questão.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
public java.lang.String getProcedureTerm()  
```  
  
## <a name="return-value"></a>Valor de retorno  
 Um **cadeia de caracteres** que contém o termo do procedimento.  
  
## <a name="exceptions"></a>Exceções  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Comentários  
 Esse método getProcedureTerm é especificado pelo método getProcedureTerm na interface DatabaseMetadata.

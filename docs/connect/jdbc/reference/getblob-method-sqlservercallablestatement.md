# <a name="getblob-method-sqlservercallablestatement"></a>Método getBlob (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera o valor do parâmetro JDBC BLOB designado como um objeto de Blob na linguagem de programação Java.  
  
## <a name="overload-list"></a>Lista de sobrecargas  
  
|Nome|Description|  
|----------|-----------------|  
|[getBlob (int)](../../../connect/jdbc/reference/getblob-method-int.md)|Recupera o valor do parâmetro JDBC BLOB designado como um objeto de Blob na linguagem de programação Java considerando o índice do parâmetro.|  
|[getBlob (Java)](../../../connect/jdbc/reference/getblob-method-java-lang-string.md)|Recupera o valor do parâmetro JDBC BLOB designado como um objeto de Blob na linguagem de programação Java considerando o nome do parâmetro.|

# <a name="getconnection-method-sqlserverstatement"></a>getConnection método (SQLServerStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera o [SQLServerConnection](../../../connect/jdbc/reference/sqlserverconnection-class.md) objeto que gerou esse [SQLServerStatement](../../../connect/jdbc/reference/sqlserverstatement-class.md) objeto.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
public final java.sql.Connection getConnection()  
```  
  
## <a name="exceptions"></a>Exceções  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Remarks  
 Esse método getConnection é especificado pelo método getConnection na interface Java.SQL. Statement.

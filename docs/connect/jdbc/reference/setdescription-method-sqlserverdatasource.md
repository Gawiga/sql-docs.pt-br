# <a name="setdescription-method-sqlserverdatasource"></a>Método setDescription (SQLServerDataSource)

## <a name="syntax"></a>Sintaxe  
  
```  
  
public void setDescription(java.lang.String description)  
```  
  
#### <a name="parameters"></a>Parâmetros  
 *Descrição*  
  
 Um **cadeia de caracteres** que contém a descrição.

# <a name="getinstancename-method-sqlserverdatasource"></a>Método getInstanceName (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Retorna o [!INCLUDE[ssNoVersion](../../../includes/ssnoversion_md.md)] nome da instância.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
public java.lang.String getInstanceName()  
```  
  
## <a name="return-value"></a>Valor de retorno  
 Um **cadeia de caracteres** que contém a instância de nome, ou nula se nenhum valor for definido.

# <a name="createnclob-method-sqlserverconnection"></a>Método createNClob (SQLServerConnection)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Cria um objeto NClob sem dados.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
public java.sql.NClob createNClob()  
```  
  
## <a name="return-value"></a>Valor retornado  
 Um objeto NClob.  
  
## <a name="exceptions"></a>Exceções  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Remarks  
 Esse método createNClob é especificado pelo método na interface do Connection createNClob.

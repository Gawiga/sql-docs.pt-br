# <a name="createnclob-method-sqlserverconnection"></a>Método createNClob (SQLServerConnection)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Cria um objeto NClob sem dados.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
public java.sql.NClob createNClob()  
```  
  
## <a name="return-value"></a>Valor de retorno  
 Um objeto NClob.  
  
## <a name="exceptions"></a>Exceções  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Comentários  
 Esse método createNClob é especificado pelo método createNClob na interface Java.SQL.

# <a name="updatestring-method-sqlserverresultset"></a>updateString método (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

|[updateString (int, Java)](../../../connect/jdbc/reference/updatestring-method-int-java-lang-string.md)|Atualiza a coluna designada com um **cadeia de caracteres** valor, considerando o índice da coluna.|

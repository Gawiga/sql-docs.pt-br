# <a name="getdrivername-method-sqlserverdatabasemetadata"></a>Método getDriverName (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera o nome do driver JDBC em questão.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
public java.lang.String getDriverName()  
```  
  
## <a name="return-value"></a>Valor retornado  
 Umaque contém o nome do driver JDBC.  
  
## <a name="exceptions"></a>Exceções  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Remarks  
 Esse método getDriverName é especificado pelo método getDriverName na interface DatabaseMetadata.

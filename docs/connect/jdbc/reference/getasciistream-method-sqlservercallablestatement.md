# <a name="getasciistream-method-sqlservercallablestatement"></a>Método getAsciiStream (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera o valor do parâmetro designado como um fluxo de **ASCII** caracteres.  
  
## <a name="overload-list"></a>Lista de sobrecargas  
  
|Nome|Description|  
|----------|-----------------|  
|[getAsciiStream &#40; int &#41;](../../../connect/jdbc/reference/getasciistream-int.md)|Recupera o valor do parâmetro designado como um fluxo de **ASCII** caracteres, considerando o índice do parâmetro.|  
|[getAsciiStream #40;java.lang.String &#41;](../../../connect/jdbc/reference/getasciistream-java-lang-string.md)|Recupera o valor do parâmetro designado como um fluxo de **ASCII** caracteres, considerando o nome do parâmetro.|

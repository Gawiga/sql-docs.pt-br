# <a name="typessservercursorforwardonly-field-sqlserverresultset"></a>Campo TYPE_SS_SERVER_CURSOR_FORWARD_ONLY (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Usado para especificar um [!INCLUDE[ssNoVersion](../../../includes/ssnoversion_md.md)] tipo de cursor de somente avanço, somente leitura.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
public static final int TYPE_SS_SERVER_CURSOR_FORWARD_ONLY  
```  
  
## <a name="field-value"></a>Valor do campo  
 Um **int** valor de 2004.

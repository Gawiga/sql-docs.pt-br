# <a name="length-method-sqlserverclob"></a>Método length (SQLServerClob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Retorna o número de caracteres em um CLOB.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
public long length()  
```  
  
## <a name="return-value"></a>Valor de retorno  
 O comprimento do CLOB em número de caracteres.  
  
## <a name="exceptions"></a>Exceções  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Remarks  
 Esse método de comprimento é especificado pelo método na interface do CLOB comprimento.

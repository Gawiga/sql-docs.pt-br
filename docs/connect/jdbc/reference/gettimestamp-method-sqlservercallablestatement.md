# <a name="gettimestamp-method-sqlservercallablestatement"></a>Método getTimestamp (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

|[getTimestamp (int, java.util.Calendar)](../../../connect/jdbc/reference/gettimestamp-method-int-java-util-calendar.md)|Recupera o valor do parâmetro designado como um objeto Java.SQL. timestamp na linguagem de programação Java considerando o índice do parâmetro, usando um objeto de calendário.|  
|[getTimestamp (Java)](../../../connect/jdbc/reference/gettimestamp-method-java-lang-string.md)|Recupera o valor do parâmetro designado como um objeto java.sql.Timestamp na linguagem de programação Java, considerando o nome do parâmetro.|  
|[getTimestamp (Java, java.util.Calendar)](../../../connect/jdbc/reference/gettimestamp-method-java-lang-string-java-util-calendar.md)|Recupera o valor do parâmetro designado como um objeto Java.SQL. timestamp na linguagem de programação Java considerando o nome do parâmetro, usando um objeto de calendário.|

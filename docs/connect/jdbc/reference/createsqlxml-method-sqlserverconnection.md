# <a name="createsqlxml-method-sqlserverconnection"></a>Método createSQLXML (SQLServerConnection)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Cria uma **SQLXML** objeto sem dados.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
public java.sql.SQLXML createSQLXML()  
```  
  
## <a name="return-value"></a>Valor retornado  
 Um objeto SQLXML.  
  
## <a name="exceptions"></a>Exceções  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Remarks  
 Esse método createSQLXML é especificado pelo método createSQLXML na interface do Connection.

# <a name="getdescription-method-sqlserverdatasource"></a>Método getDescription (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Retorna uma descrição da fonte de dados.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
public java.lang.String getDescription()  
```  
  
## <a name="return-value"></a>Valor de retorno  
 Um **cadeia de caracteres** que contém a descrição da fonte de dados ou nulo se nenhum valor for definido.

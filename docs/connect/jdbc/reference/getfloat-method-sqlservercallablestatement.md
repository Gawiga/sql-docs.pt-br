# <a name="getfloat-method-sqlservercallablestatement"></a>Método getFloat (SQLServerCallableStatement)

|----------|-----------------|  
|[getFloat (int)](../../../connect/jdbc/reference/getfloat-method-int.md)|Recupera o valor do parâmetro designado como um **float** em considerando o índice do parâmetro de linguagem de programação Java.|

# <a name="getfetchsize-method-sqlserverresultset"></a>getFetchSize método (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera o tamanho da busca para este [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) objeto.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
public int getFetchSize()  
```  
  
## <a name="return-value"></a>Valor de retorno  
 Um **int** que indica o tamanho da busca atual.  
  
## <a name="exceptions"></a>Exceções  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Comentários  
 Esse método getFetchSize é especificado pelo método getFetchSize na interface Java.SQL. resultset.

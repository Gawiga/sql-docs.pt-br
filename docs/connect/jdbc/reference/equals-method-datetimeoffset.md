# <a name="equals-method-datetimeoffset"></a>Método equals (DateTimeOffset)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Compara este **DateTimeOffset** objeto para outro **objeto**.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
public boolean equals(Object o)  
```  
  
#### <a name="parameters"></a>Parâmetros  
 *o*  
  
 O objeto que você deseja comparar com a instância atual.  
  
## <a name="return-value"></a>Valor de retorno  
 Retorna VERDADEIRO se este **DateTimeOffset** objeto representa o mesmo ponto no tempo no mesmo deslocamento do GMT como outro **DateTimeOffset**.

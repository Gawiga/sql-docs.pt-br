# <a name="getobject-method-sqlserverresultset"></a>getObject método (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Obtém o valor da coluna designada na linha atual deste [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) objeto como um objeto na linguagem de programação Java.  
  
## <a name="overload-list"></a>Lista de sobrecargas  
  
|Nome|Description|  
|----------|-----------------|  
|[getObject (int)](../../../connect/jdbc/reference/getobject-method-int-sqlserverresultset.md)|Obtém o valor do índice de coluna designada na linha atual deste [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) objeto como um objeto na linguagem de programação Java.|  
|[getObject (int, java.util.Map)](../../../connect/jdbc/reference/getobject-method-int-java-util-map-sqlserverresultset.md)|Obtém o valor do índice de coluna designada na linha atual deste [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) objeto como um objeto em Java linguagem de programação, usando o objeto de mapa fornecido.|  
|[getObject (Java)](../../../connect/jdbc/reference/getobject-method-java-lang-string-sqlserverresultset.md)|Obtém o valor do nome da coluna designada na linha atual deste [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) objeto como um objeto na linguagem de programação Java.|  
|[getObject (Java, java.util.Map)](../../../connect/jdbc/reference/getobject-method-java-lang-string-java-util-map-sqlserverresultset.md)|Obtém o valor do nome da coluna designada na linha atual deste [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) objeto como um objeto em Java linguagem de programação, usando o objeto de mapa fornecido.|

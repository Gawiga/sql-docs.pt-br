# <a name="getblob-method-sqlserverresultset"></a>getBlob método (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera o valor da coluna designada na linha atual deste [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) objeto como um objeto de Blob na linguagem de programação Java.  
  
## <a name="overload-list"></a>Lista de sobrecargas  
  
|Nome|Description|  
|----------|-----------------|  
|[getBlob (int)](../../../connect/jdbc/reference/getblob-method-int-sqlserverresultset.md)|Recupera o valor do índice de coluna designada na linha atual deste [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) objeto como um objeto de Blob na linguagem de programação Java.|  
|[getBlob (Java)](../../../connect/jdbc/reference/getblob-method-java-lang-string-sqlserverresultset.md)|Recupera o valor do nome da coluna designada na linha atual deste [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) objeto como um objeto de Blob na linguagem de programação Java.|

# <a name="getshort-method-sqlservercallablestatement"></a>Método getShort (SQLServerCallableStatement)

|Nome|Description|  
|----------|-----------------|  
|[getShort (int)](../../../connect/jdbc/reference/getshort-method-int.md)|Recupera o valor do parâmetro designado como um **curto** em considerando o índice do parâmetro de linguagem de programação Java.|

# <a name="setpassword-method-sqlserverdatasource"></a>Método setPassword (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Define a senha que será usada para se conectar ao [!INCLUDE[ssNoVersion](../../../includes/ssnoversion_md.md)].  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
public void setPassword(java.lang.String password)  
```  
  
#### <a name="parameters"></a>Parâmetros  
 *senha*  
  
 Um **cadeia de caracteres** que contém a senha.

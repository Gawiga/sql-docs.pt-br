# <a name="getcharacterstream-method-sqlserverclob"></a>Método getCharacterStream (SQLServerClob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera o **Clob** valor designado pelo objeto CLOB como um objeto Java.IO. Reader.  
  
## <a name="overload-list"></a>Lista de sobrecargas  
  
|Nome|Description|  
|----------|-----------------|  
|[Método getCharacterStream &#40; &#41;](../../../connect/jdbc/reference/getcharacterstream-method.md)|Retorna os dados de Clob como um objeto java.io.Reader ou como um fluxo de caracteres.|  
|[getCharacterStream método &#40; long, long &#41;](../../../connect/jdbc/reference/getcharacterstream-method-long-long.md)|Retorna os dados de Clob como um objeto Java.IO. Reader ou como um fluxo de caracteres com o tamanho e a posição especificada.|

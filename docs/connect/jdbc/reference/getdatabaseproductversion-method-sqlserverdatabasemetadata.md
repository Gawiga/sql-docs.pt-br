# <a name="getdatabaseproductversion-method-sqlserverdatabasemetadata"></a>Método getDatabaseProductVersion (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera o número da versão do produto de banco de dados em questão.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
public java.lang.String getDatabaseProductVersion()  
```  
  
## <a name="return-value"></a>Valor de retorno  
 Um **cadeia de caracteres** que contém o número de versão de produto do banco de dados.  
  
## <a name="exceptions"></a>Exceções  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Remarks  
 Esse método getDatabaseProductVersion é especificado pelo método getDatabaseProductVersion na interface DatabaseMetadata.

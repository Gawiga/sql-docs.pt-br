# <a name="sqlserverdriver-class"></a>Classe SQLServerDriver
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

 **Pacote:** com.microsoft.sqlserver.jdbc  
  
 **Estende:** java.lang.Object  
  
 **Implementa:** Java.SQL. driver

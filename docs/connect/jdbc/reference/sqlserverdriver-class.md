# <a name="sqlserverdriver-class"></a>Classe SQLServerDriver
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Representa o [!INCLUDE[jdbcNoVersion](../../../includes/jdbcnoversion_md.md)]. Essa classe inclui métodos para se conectar a um banco de dados do [!INCLUDE[ssNoVersion](../../../includes/ssnoversion_md.md)] e para obter informações sobre o driver JDBC.  
  
 **Pacote:** com.microsoft.sqlserver.jdbc  
  
 **Estende:** java.lang.Object  
  
 **Implementa:** java.sql.Driver  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
public final class SQLServerDriver  
```

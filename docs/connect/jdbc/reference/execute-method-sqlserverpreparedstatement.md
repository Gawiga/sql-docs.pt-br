# <a name="execute-method-sqlserverpreparedstatement"></a>Executar método (SQLServerPreparedStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Executa a instrução SQL na [SQLServerPreparedStatement](../../../connect/jdbc/reference/sqlserverpreparedstatement-class.md) objeto, que pode ser qualquer tipo de instrução SQL.  
  
## <a name="overload-list"></a>Lista de sobrecargas  
  
|Nome|Description|  
|----------|-----------------|  
|[Execute)](../../../connect/jdbc/reference/execute-method.md)|Executa a instrução SQL na [SQLServerPreparedStatement](../../../connect/jdbc/reference/sqlserverpreparedstatement-class.md) objeto, que pode ser qualquer tipo de instrução SQL.|  
|[Execute (Java)](../../../connect/jdbc/reference/execute-method-java-lang-string.md)|Executa a instrução SQL fornecida que pode retornar vários resultados.|

# <a name="concurssoptimisticccval-field-sqlserverresultset"></a>Campo CONCUR_SS_OPTIMISTIC_CCVAL (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Usado para especificar um tipo de simultaneidade otimista de leitura/gravação do [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] sem bloqueios de linha.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
public static final int CONCUR_SS_OPTIMISTIC_CCVAL  
```  
  
## <a name="field-value"></a>Valor do campo  
 Uma **int** valor 1010.

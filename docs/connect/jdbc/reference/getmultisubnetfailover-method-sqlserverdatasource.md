# <a name="getmultisubnetfailover-method-sqlserverdatasource"></a>Método getMultiSubnetFailover (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Retorna o valor da **multiSubnetFailover** propriedade de conexão.  
  
## <a name="syntax"></a>Sintaxe  
  
```vb  
public boolean getMultiSubnetFailover();  
```  
  
## <a name="return-value"></a>Valor de retorno  
 Retorna true ou false, dependendo da configuração atual da propriedade de conexão.  
  
## <a name="remarks"></a>Comentários  
 Para obter mais informações sobre o **multiSubnetFailover** propriedade de conexão, consulte [definindo as propriedades de Conexão](../../../connect/jdbc/setting-the-connection-properties.md).

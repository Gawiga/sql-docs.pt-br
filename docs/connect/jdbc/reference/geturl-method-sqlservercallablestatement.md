# <a name="geturl-method-sqlservercallablestatement"></a>Método getURL (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

|Nome|Description|  
|----------|-----------------|  
|[getURL (int)](../../../connect/jdbc/reference/geturl-method-int.md)|Recupera o valor do parâmetro designado como um objeto na linguagem de programação Java considerando o índice do parâmetro de URL.|  
|[getURL (Java)](../../../connect/jdbc/reference/geturl-method-java-lang-string.md)|Recupera o valor do parâmetro designado como um objeto na linguagem de programação Java considerando o nome do parâmetro de URL.|

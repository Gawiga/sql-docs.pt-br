# <a name="redistributing-the-microsoft-jdbc-driver"></a>Redistribuindo o Microsoft JDBC Driver
[!INCLUDE[Driver_JDBC_Download](../../includes/driver_jdbc_download.md)]

    
## <a name="register-to-redistribute-the-microsoft-jdbc-40-driver"></a>Registre-se para redistribuir o Microsoft JDBC 4.0 Driver  
 Clique no link abaixo para examinar o contrato de licença para a versão 4.0 do driver.  Se quiser, você poderá imprimir e reter uma cópia do contrato de licença para seu controle.  
  
 Se optar por aceitar o contrato de licença, você será direcionado para a página de registro e, em seguida, para a página de download.  
  
 [Licença de redistribuição do Microsoft JDBC Driver 4.0 para SQL Server](https://msdn.microsoft.com/sqlserver/jj589698)  
  
 Observação: a Microsoft usa o Windows Live ID para autenticação e registro seguros. Durante o seu registro para obter direitos de redistribuição para o SQL Server JDBC Driver usando o Windows Live ID, você precisará fornecer informações que se tornarão parte do seu perfil seguro.  
  
 Você pode obter um  [Windows Live ID](https://signup.live.com/), se ainda não tiver um  
  

## <a name="redistribute-the-microsoft-jdbc-41-42-and-60-driver"></a>Redistribuir o Driver JDBC do Microsoft 6.0, 4.2 e 4.1
Analise a cláusula "Código distribuível" nos contratos de licença para o 4.1, 4.2, 6.0 e 6.2 versões do driver.

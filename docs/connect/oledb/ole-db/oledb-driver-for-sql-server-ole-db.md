# <a name="ole-db-driver-for-sql-server-ole-db"></a>Driver do OLE DB para SQL Server (OLE DB)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]
  O Driver OLE DB para SQL Server é uma API COM baixo nível que é usado para acessar dados. O Driver OLE DB para SQL Server é recomendado para desenvolver ferramentas, utilitários ou componentes de baixo nível que precisam de alto desempenho. O Driver OLE DB para SQL Server é um provedor nativo de alto desempenho que acessa o [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] fluxo TDS (Tabular Data) do protocolo diretamente.  
  
 OLE DB Driver para SQL Server fornece suporte de OLE DB a aplicativos que se conectam ao [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].  
  
 O Driver OLE DB para SQL Server é um provedor do OLE DB versão 2.0 – compatíveis.

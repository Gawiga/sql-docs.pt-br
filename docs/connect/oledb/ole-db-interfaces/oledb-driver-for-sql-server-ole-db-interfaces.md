# <a name="ole-db-driver-for-sql-server-ole-db-interfaces"></a>Driver do OLE DB para Interfaces do SQL Server (OLE DB)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]

  O Driver OLE DB para SQL Server expõe as interfaces para consumidores que desejam acesso a dados em um ou mais computadores que executam o [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]. O Driver OLE DB para SQL Server é um provedor do OLE DB versão 2.0 – compatíveis.  
  
 Esse Driver OLE DB para referência de programação do SQL Server não documenta todas as interfaces do OLE DB e métodos, somente essas interfaces e métodos que exibam um comportamento específico do provedor ao usar o Driver OLE DB para SQL Server.  
  
 Para obter uma descrição completa das interfaces do OLE DB de núcleo e métodos, consulte o [referência do programador de DB OLE](http://go.microsoft.com/fwlink/?LinkId=45232).

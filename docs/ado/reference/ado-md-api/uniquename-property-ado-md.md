# <a name="uniquename-property-ado-md"></a>Propriedade UniqueName (ADO MD)
Indica o nome ambíguo para o objeto atual.  
  
## <a name="return-values"></a>Valores de retorno  
 Retorna um **cadeia de caracteres** e é somente leitura.

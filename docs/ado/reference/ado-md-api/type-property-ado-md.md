# <a name="type-property-ado-md"></a>Propriedade Type (ADO MD)
Indica o tipo de atual [membro](../../../ado/reference/ado-md-api/member-object-ado-md.md).  
  
## <a name="return-values"></a>Valores de retorno  
 Retorna um [MemberTypeEnum](../../../ado/reference/ado-md-api/membertypeenum.md) valor e é somente leitura.  
  
## <a name="remarks"></a>Remarks  
 Essa propriedade é suportada apenas em [membro](../../../ado/reference/ado-md-api/member-object-ado-md.md) objetos que pertencem a um [nível](../../../ado/reference/ado-md-api/level-object-ado-md.md) objeto. Ocorre um erro quando essa propriedade é referenciada em **membro** objetos que pertencem a um [posição](../../../ado/reference/ado-md-api/position-object-ado-md.md) objeto.

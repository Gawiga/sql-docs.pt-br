# <a name="attributes-property-adox"></a>Propriedade Attributes (ADOX)
Descreve as características da coluna.  
  
## <a name="settings-and-return-values"></a>Configurações e valores de retorno  
 Define ou retorna um **longo** valor. O valor Especifica as características da tabela que é representado pelo [coluna](../../../ado/reference/adox-api/column-object-adox.md) objeto. O valor pode ser uma combinação de [ColumnAttributesEnum](../../../ado/reference/adox-api/columnattributesenum.md) constantes. O valor padrão é zero (**0**), que não é nem **adColFixed** nem **adColNullable**.  
  
## <a name="applies-to"></a>Aplica-se a  
  
- [Objeto Column (ADOX)](../../../ado/reference/adox-api/column-object-adox.md)

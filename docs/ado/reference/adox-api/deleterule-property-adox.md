# <a name="deleterule-property-adox"></a>Propriedade DeleteRule (ADOX)
Indica a ação executada quando uma chave primária é excluída.  
  
## <a name="settings-and-return-values"></a>Configurações e valores de retorno  
 Define e retorna um **longo** que pode ser um valor de [RuleEnum](../../../ado/reference/adox-api/ruleenum.md) constantes. O valor padrão é **adRINone**.  
  
## <a name="remarks"></a>Remarks  
 Essa propriedade é somente leitura no [chave](../../../ado/reference/adox-api/key-object-adox.md) já está anexados a uma coleção de objetos.

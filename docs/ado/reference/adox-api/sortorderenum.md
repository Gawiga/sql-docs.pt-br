# <a name="sortorderenum"></a>SortOrderEnum
Especifica a sequência de classificação para uma coluna indexada.  
  
|Constante|Value|Description|  
|--------------|-----------|-----------------|  
|**adSortAscending**|1|Padrão. A sequência de classificação para a coluna em ordem crescente.|  
|**adSortDescending**|2|A sequência de classificação para a coluna em ordem decrescente.|

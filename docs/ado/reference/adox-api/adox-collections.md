|||  
|-|-|  
|[Colunas](../../../ado/reference/adox-api/columns-collection-adox.md)|Contém todos os **coluna** objetos de tabela, índice ou chave.|  
|[Grupos](../../../ado/reference/adox-api/groups-collection-adox.md)|Contém todos os armazenados **grupo** objetos de um catálogo ou usuário.|

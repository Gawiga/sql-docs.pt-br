# <a name="sortorder-property-adox"></a>Propriedade SortOrder (ADOX)
Indica a sequência de classificação da coluna (somente colunas de índice).  
  
## <a name="settings-and-return-values"></a>Configurações e valores de retorno  
 Define e retorna um **longo** que pode ser um valor de [SortOrderEnum](../../../ado/reference/adox-api/sortorderenum.md) constantes. O valor padrão é **adSortAscending**.  
  
## <a name="remarks"></a>Remarks  
 Essa propriedade só se aplica a [coluna](../../../ado/reference/adox-api/column-object-adox.md) objetos no [colunas](../../../ado/reference/adox-api/columns-collection-adox.md) coleção de um [índice](../../../ado/reference/adox-api/index-object-adox.md).

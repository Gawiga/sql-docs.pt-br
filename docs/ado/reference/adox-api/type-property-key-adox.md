# <a name="type-property-key-adox"></a>Propriedade de tipo (chave) (ADOX)
Indica o tipo da chave.  
  
## <a name="settings-and-return-values"></a>Configurações e valores de retorno  
 Define ou retorna um **longo** valor que pode ser uma da [KeyTypeEnum](../../../ado/reference/adox-api/keytypeenum.md) constantes. O valor padrão é **adKeyPrimary**.  
  
## <a name="remarks"></a>Comentários  
 Essa propriedade é somente leitura no [chave](../../../ado/reference/adox-api/key-object-adox.md) já está anexados a uma coleção de objetos.

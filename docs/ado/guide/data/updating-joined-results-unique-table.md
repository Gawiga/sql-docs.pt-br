# <a name="updating-joined-results-unique-table"></a>Atualizando Unido resultados: Tabela exclusiva
ADO permite atentamente as modificações de controle para uma determinada tabela base em um conjunto de registros foi formado por uma operação de junção em várias tabelas base usando a propriedade dinâmica da tabela exclusiva. Para obter detalhes sobre como usar a tabela exclusiva, consulte os tópicos de referência do programador do ADO sobre as propriedades dinâmicas da tabela exclusiva e atualização de ressincronização.

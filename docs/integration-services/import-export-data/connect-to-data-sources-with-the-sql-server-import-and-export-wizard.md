# <a name="connect-to-data-sources-with-the-sql-server-import-and-export-wizard"></a>Conecte-se a fontes de dados com o Assistente de exportação e importação do SQL Server
Os tópicos nesta seção mostram como se conectar a várias fontes de dados usadas quando você executa o Assistente para exportação e importação do SQL Server. Você precisa fornecer informações de conexão em suas fontes de dados de **escolher uma fonte de dados** e **escolha um destino** páginas do assistente.

Os tópicos nesta seção descrevem como somente para **se conectar a fontes de dados** do **escolher uma fonte de dados** e **escolha um destino** páginas do assistente. Se você estiver procurando por alguma outra coisa, consulte [relacionadas a tarefas e conteúdo](#related).

# <a name="sql-server-2012-sp3-release-notes"></a>Notas de Versão do SQL Server 2012 SP3
Estas notas de versão descrevem problemas que você precisa saber antes de instalar ou solucionar problemas de SQL Server 2012 Service Pack 3. As Notas de Versão listadas aqui estão disponíveis somente online, e não em mídia de instalação. Elas são atualizadas periodicamente, conforme os problemas são descobertos. Para obter uma lista o melhorias e correções, consulte [informações de versão do SQL Server 2012 Service Pack 3](http://support.microsoft.com/help/3072779/sql-server-2012-service-pack-3-release-information) para obter mais informações.  
  
## <a name="choose-the-correct-file-to-download-and-install"></a>Escolha o arquivo correto para baixar e instalar  
Use a tabela abaixo para identificar o local e o nome do arquivo a ser baixado com base em sua versão atualmente instalada. As páginas de download têm requisitos de sistema e instruções básicas de instalação.  

##<a name="download-pages"></a>Páginas de download
O link a seguir para principal baixar pacotes para o SQL Server 2012 SP3.
- [SQL Server 2012 SP3](http://go.microsoft.com/fwlink/?linkid=615935)
- [SQL Server 2012 SP3 Express](http://go.microsoft.com/fwlink/?linkid=692144)
- [Pacote de recursos do Microsoft SQL Server 2012 SP3](http://go.microsoft.com/fwlink/?linkid=615941)

Para obter mais informações para identificar o local e o nome do arquivo a ser baixado com base na sua versão atualmente instalada, consulte a seção "Selecionar o arquivo correto para baixar" [informações de versão do SQL Server 2012 Service Pack 3](https://support.microsoft.com/en-us/help/3072779/sql-server-2012-service-pack-3-release-information).

# <a name="position-object-ado-md"></a>Objeto de posição (ADO MD)
Representa um conjunto de um ou mais membros de diferentes dimensões que define um ponto em um eixo.  
  
## <a name="remarks"></a>Remarks  
 Com as propriedades e coleções de uma **posição** objeto pode fazer o seguinte:  
  
-   Use o **Ordinal** propriedade para retornar a posição ordinal do **posição** ao longo do [eixo](../../../ado/reference/ado-md-api/axis-object-ado-md.md).  
  
-   Use o [membros](../../../ado/reference/ado-md-api/members-collection-ado-md.md) coleção para retornar os membros que formam a posição ao longo de **eixo**.

# <a name="parentsameasprev-property-ado-md"></a>Propriedade ParentSameAsPrev (ADO MD)
Indica se o pai nessa posição [membro](../../../ado/reference/ado-md-api/member-object-ado-md.md) é o mesmo que o pai do membro imediatamente anterior.  
  
## <a name="return-values"></a>Valores de retorno  
 Retorna um **booliano** valor e é somente leitura.  
  
## <a name="remarks"></a>Remarks  
 Essa propriedade é suportada apenas em [membro](../../../ado/reference/ado-md-api/member-object-ado-md.md) objetos que pertencem a um [posição](../../../ado/reference/ado-md-api/position-object-ado-md.md) objeto. Ocorre um erro quando essa propriedade é referenciada em **membro** objetos que pertencem a um [nível](../../../ado/reference/ado-md-api/level-object-ado-md.md) objeto.  
  
## <a name="applies-to"></a>Aplica-se a  
 [Objeto Member (ADO MD)](../../../ado/reference/ado-md-api/member-object-ado-md.md)

# <a name="getrowsoptionenum"></a>GetRowsOptionEnum
Especifica quantos registros para recuperar um [registros](../../../ado/reference/ado-api/recordset-object-ado.md).  
  
|Constante|Value|Description|  
|--------------|-----------|-----------------|  
|**adGetRowsRest**|-1|Recupera o restante dos registros da **registros**, a partir da posição atual ou um indicador especificado pelo *iniciar* parâmetro do [GetRows](../../../ado/reference/ado-api/getrows-method-ado.md) método.|

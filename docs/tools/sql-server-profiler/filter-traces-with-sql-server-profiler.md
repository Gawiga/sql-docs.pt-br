# <a name="filter-traces-with-sql-server-profiler"></a>Filtrar rastreamentos com o SQL Server Profiler
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]Você pode definir filtros em colunas de dados para [!INCLUDE[ssSqlProfiler](../../includes/sssqlprofiler-md.md)] eventos de rastreamento para que sejam capturadas informações apenas para os eventos que você está interessado. Os filtros de rastreamento reduzem o impacto de rastrear o desempenho de sistema porque menos eventos são capturados no rastreamento. A filtragem também torna a saída do rastreamento mais fácil de ler, pois o número de eventos é menor.

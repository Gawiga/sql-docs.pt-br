# <a name="information-to-obtain-from-your-ihv"></a>Informações para obter de sua IHVS
Quando o fornecedor do hardware independentes (IHVS) fornece o novo dispositivo de PDW do SQL Server para você, eles também fornecerá informações sobre o hardware do dispositivo e a configuração realizou no seu dispositivo. Você precisará dessas informações para administrar seu dispositivo.  
  
A lista a seguir mostra informações que normalmente é necessário do seu IHVS. Em alguns casos, informações adicionais ou é necessário. Verifique com seu IHVS para certificar-se de que todas as informações relevantes foi transferidas para você com a entrega do dispositivo.  
  
|||  
|-|-|  
|**Informações ou documento**|**Description**|  
|Lista de materiais (BOM)|Lista de materiais lista os componentes que estão incluídos em seu dispositivo. Essas informações são necessárias para confirmar que todos os componentes foram entregues.<br /><br />**Importante:** sua lista de materiais deve incluir os pesos para cada um de nós de dispositivo e para cada rack completo. Essas informações são importantes ao planejar como tratar e mover componentes do dispositivo e para verificar se o seu data center pode acomodar o dispositivo. Se o seu BOM não inclui os pesos de nó, certifique-se de obter essas informações de seu IHVS para todos os nós.|  
|Diagramas de cabeamento|Cabeamento diagramas mostram como conectar-se a rede, energia e outros cabos para cada dispositivo do rack. Esses diagramas são necessários para instalar o dispositivo no seu data center e sempre que você precisa remover ou substituir um componente.|  
|Requisitos de posicionamento no rack do dispositivo|Antes de seu dispositivo pode ser instalado em seu data center, você precisa saber se o seu data center atende o fluxo de ar e requisitos de comprimento de cabo para o dispositivo, bem como o tamanho e os requisitos de energia para os componentes. Consulte também lista de materiais (BOM) acima para obter informações sobre os pesos de componente do dispositivo, que também é necessário.|

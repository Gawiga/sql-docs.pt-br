# <a name="pdw-topology"></a>Topologia do PDW
Exibe o nome e a rede informações sobre os componentes do SQL Server PDW. Use o **configuração de rede de topologia do dispositivo** página para alterar a endereços IP de rede.  
  
Expanda os elementos desta seção para configurar informações que se aplicam ao dispositivo inteiro. Selecione as seções a topologia de depósito de dados paralela ou a topologia de HDInsight para configurar informações que se aplica a essa região do dispositivo.

# <a name="tutorials-for-sql-server-2016"></a>Tutoriais do SQL Server 2016
[!INCLUDE[feedback_stackoverflow_msdn_connect-md](../includes/feedback-stackoverflow-msdn-connect-md.md)]

[!INCLUDE[ssCurrent](../includes/sscurrent-md.md)] incorpora várias tecnologias e recursos novos. Três tutoriais novos ou significativamente atualizados estão disponíveis para ajudar você a começar a trabalhar com o [!INCLUDE[ssCurrent](../includes/sscurrent-md.md)]sem nenhuma dificuldade.  
  
Há vários planos que visam fornecer tutoriais adicionais no futuro. Enquanto isso, você pode continuar aprendendo com os tutoriais publicados para versões anteriores. Em alguns casos, você pode encontrar diferenças secundárias entre a interface do usuário do [!INCLUDE[ssCurrent](../includes/sscurrent-md.md)] e das versões anteriores ou talvez haja diferenças sutis nos pré-requisitos do tutorial. Se um tutorial usar um dos bancos de dados do [!INCLUDE[ssSampleDBCoShort](../includes/sssampledbcoshort-md.md)] , você deverá baixar e usar a versão [!INCLUDE[ssCurrent](../includes/sscurrent-md.md)] .  
  
Se você estiver usando um servidor do [!INCLUDE[ssCurrent](../includes/sscurrent-md.md)] , os provedores de dados do [!INCLUDE[ssCurrent](../includes/sscurrent-md.md)] estarão instalados. Para que os tutoriais funcionem, você precisará dos bancos de dados de exemplo do [!INCLUDE[ssCurrent](../includes/sscurrent-md.md)] que funcionam com esses provedores ou dos provedores de dados que funcionam com a versão anterior dos bancos de dados de exemplo.  
  

  
-   Baixe uma pasta de trabalho do [!INCLUDE[ssGemini](../includes/ssgemini-md.md)] e os relatórios do [!INCLUDE[ssCrescent](../includes/sscrescent-md.md)] a serem usados com os tutoriais do [!INCLUDE[ssCrescent](../includes/sscrescent-md.md)] : vá para [Amostras do Power View para SQL Server 2012](http://go.microsoft.com/fwlink/?LinkId=220734).  
- Confira a nova amostra [Wide World Importers](https://msdn.microsoft.com/library/mt734199(SQL.1).aspx) disponível no repositório GitHub [sql-server-samples](https://github.com/Microsoft/sql-server-samples) . 
    
-   Baixe versões do [!INCLUDE[ssCurrent](../includes/sscurrent-md.md)] de bancos de dados [!INCLUDE[ssSampleDBCoShort](../includes/sssampledbcoshort-md.md)] : vá para [Bancos de dados AdventureWorks e scripts para SQL Server 2016 CTP3](https://www.microsoft.com/download/details.aspx?id=49502) no CodePlex.

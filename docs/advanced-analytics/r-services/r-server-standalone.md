# R Server (Aut&#244;nomo)
  **Microsoft R servidor (autônomo)** faz parte da plataforma de análise empresarial fornecida pelo Microsoft R Services.  
  
 Ele estende R fornecendo suporte, escalabilidade e segurança. [!INCLUDE[rsql_platform](../../includes/rsql-platform-md.md)] também aborda as limitações de memória do código-fonte aberto R adicionando paralelo e processamento em partes de dados, permitindo aos usuários analisar dados em escala empresarial e usar dados muito maiores do que pode caber na memória.  
  
  
## Introdução ao Microsoft R Server  
  
 Para obter mais informações sobre como configurar um servidor do Windows para oferecer suporte a aplicativos de R, consulte [criar um Standalone R Server](../../advanced-analytics/r-services/create-a-standalone-r-server.md).  

Você também pode provisionar uma máquina virtual do Azure que inclui um servidor de R (autônomo) junto com DeployR, para incorporar analytics R dentro de aplicativos e integração de R com sistemas de back-end. Para obter mais informações, consulte:

+ [Provisionar uma máquina Virtual do servidor R](../../advanced-analytics/r-services/provision-the-r-server-only-sql-server-2016-enterprise-vm-on-azure.md)
+ [Implantar R](https://msdn.microsoft.com/microsoft-r/deployr-about)
  
Para obter mais informações sobre o Microsoft R, tutoriais e exemplos, consulte [produtos da Microsoft R](https://msdn.microsoft.com/microsoft-r/microsoft-r-getting-started).

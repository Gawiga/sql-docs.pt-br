# <a name="monitoring-python-in-sql-server"></a>Monitoramento de Python no SQL Server
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md-winonly](../../includes/appliesto-ss-xxxx-xxxx-xxx-md-winonly.md)]

Suporte para Python é um novo recurso do SQL Server, disponível começando com o SQL Server 2017.

A estrutura geral de segurança de dados e a execução do script externo é igual de R. Você pode monitorar qualquer script externo, incluindo aquele executado Python, usando a mesma estrutura que foi fornecida para a execução de script R no SQL Server 2016. Para obter mais informações, consulte [Monitoring R Solutions](../r/managing-and-monitoring-r-solutions.md).

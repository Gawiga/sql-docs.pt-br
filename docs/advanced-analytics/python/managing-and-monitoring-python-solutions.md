# <a name="monitoring"></a>Monitoramento


> [!NOTE]
> Suporte do Python é um novo recurso do SQL Server 2017 e está na versão de pré-lançamento.

Suporte para Python é um novo recurso do SQL Server, disponível no SQL Server de 2017 CTP 2.0.

No entanto, a estrutura geral de segurança de dados e a execução do script externo é igual de R. Você pode monitorar qualquer script externo, incluindo aquele executado Python, usando a mesma estrutura que foi fornecida para a execução de script R no SQL Server 2016. Para obter mais informações, consulte [Monitoring R Solutions](../r/managing-and-monitoring-r-solutions.md).

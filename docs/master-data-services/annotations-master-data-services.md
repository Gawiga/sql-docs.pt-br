# Anota&#231;&#245;es (Master Data Services)
  No [!INCLUDE[ssMDSshort](../includes/ssmdsshort-md.md)], anotações são comentários que você insere para fornecer detalhes sobre transações.  
  
 É possível anotar uma transação para fornecer informações sobre por que uma ação foi tomada. Você deve selecionar a transação para adicionar uma anotação relacionada. As anotações são exibidas junto com as transações em:  
  
-   A área funcional **Explorer** do aplicativo Web do [!INCLUDE[ssMDSmdm](../includes/ssmdsmdm-md.md)].  
  
-   A área funcional do **Gerenciamento de Versões** do aplicativo Web do [!INCLUDE[ssMDSmdm](../includes/ssmdsmdm-md.md)].  
  
-   O MDS [!INCLUDE[ssMDSXLS](../includes/ssmdsxls-md.md)].

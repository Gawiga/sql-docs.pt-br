# Alterar o endere&#231;o de email de um usu&#225;rio (Master Data Services)
  No [!INCLUDE[ssMDSshort](../includes/ssmdsshort-md.md)], você pode alterar o endereço de email para o qual são enviadas notificações e o formato padrão dos emails de notificação.  
  
## Pré-requisitos  
 Para executar esse procedimento:  
  
-   Você deve ter permissão para acessar a área funcional **Usuários e Grupos**.  
  
### Para alterar o endereço de email ou o formato de email padrão de um usuário  
  
1.  No [!INCLUDE[ssMDSmdm](../includes/ssmdsmdm-md.md)], clique em **Permissões de Usuário e Grupo**.  
  
2.  Na página **Usuários**, selecione a linha do usuário que você deseja editar.  
  
3.  Clique em **Editar usuário selecionado**.  
  
4.  Acima da guia **Geral**, clique em **Editar**.  
  
5.  Atualize o endereço de email ou o formato de email.  
  
6.  Clique em **Salvar**.

# <a name="changelog-for-the-report-viewer-controls"></a>Log de mudanças para controles do Visualizador de Relatórios

Esse log de mudanças controla as versões dos controles de WinForms e WebForms do Visualizador de Relatórios.

## <a name="15900148"></a>15.900.148
Com esta atualização
 - Correção de um bug que impedia que relatórios sem parâmetros fossem carregados por meio de Server.LoadReportDefinition

O controle [WebForms](https://www.nuget.org/packages/Microsoft.ReportingServices.ReportViewerControl.Webforms/150.900.148) do Visualizador de Relatórios
 - Dá suporte à inserção em páginas RTL (páginas que alteram o fluxo de texto usando a propriedade *direction: rtl;*).
 - Dá suporte à personalização do texto da caixa de diálogo de impressão por meio da interface de localização *IReportViewerMessages5*.
 - Suporte a acessibilidade aprimorado.

O controle [WinForms](https://www.nuget.org/packages/Microsoft.ReportingServices.ReportViewerControl.Winforms/150.900.148) do Visualizador de Relatórios
 - Correção para impressão quando um aplicativo está em execução em um modo de alto DPI.

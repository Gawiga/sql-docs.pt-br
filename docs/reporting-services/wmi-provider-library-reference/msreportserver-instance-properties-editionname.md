# <a name="msreportserverinstance-properties---editionname"></a>Propriedades de MSReportServer_Instance – EditionName
  Retorna o nome legível para o usuário da edição do servidor de relatório. Somente leitura.  
  
## <a name="syntax"></a>Sintaxe  
  
```vb  
Public Dim EditionName As String  
```  
  
```csharp  
public string EditionName;  
```  
  
## <a name="property-value"></a>Valor da propriedade  
 Uma **cadeia de caracteres** que contém um nome legível por humanos da edição do servidor de relatório.

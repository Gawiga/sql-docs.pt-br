# <a name="configurationsetting-property---databaseservername"></a>Propriedade de ConfigurationSetting – DatabaseServerName
  Especifica o nome do servidor em que o banco de dados do servidor de relatório está instalado. Leitura/gravação.  
  
## <a name="syntax"></a>Sintaxe  
  
```vb  
Public Dim DatabaseServerName As String  
```  
  
```csharp  
public string DatabaseServerName;  
```  
  
## <a name="property-values"></a>Valores da propriedade  
 Um objeto **String** que representa o nome do servidor.

# Propriedade SenderEmailAddress (WMI MSReportServer_ConfigurationSetting)
  Obtém o endereço usado para enviar e-mail do servidor de relatório. Somente leitura.  
  
## Sintaxe  
  
```vb  
Public Dim SenderEmailAddress As String  
```  
  
```csharp  
public string SenderEmailAddress;  
```  
  
## Valores da propriedade  
 Um objeto **String** somente leitura que representa o endereço de email usado pelo servidor de relatório.

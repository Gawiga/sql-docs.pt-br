# Propriedade SendUsingSMTPServer (WMI MSReportServer_ConfigurationSetting)
  Especifica se o elemento **SendUsing** na configuração de email para o servidor de relatório está definido como **true**.  
  
## Sintaxe  
  
```vb  
Public Dim SendUsingSMTPServer As Boolean  
```  
  
```csharp  
public Boolean SendUsingSMTPServer;  
```  
  
## Valores da propriedade  
 Um objeto **Boolean** somente leitura.

# Caixa de di&#225;logo Inserir Credenciais da Fonte de Dados (Construtor de Relat&#243;rios)
  Use a caixa de diálogo **Inserir Credenciais da Fonte de Dados** para fornecer suas credenciais para que você possa recuperar dados da fonte de dados.  
  
## Opções  
 **Nome de usuário**  
 Digite o nome de usuário para fazer logon na fonte de dados.  
  
 **Senha**  
 Digite a senha para fazer logon na fonte de dados.

  Esta seção descreve como programar [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Management Objects (SMO) em [!INCLUDE[msCoName](../../includes/msconame-md.md)] Visual c# .NET.
  
|Tópico|Description|  
|-----------|-----------------|

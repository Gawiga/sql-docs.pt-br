## <a name="details"></a>Detalhes  
  
|||  
|-|-|  
|Nome do produto|SQL Server|  
|ID do evento|3260|  
|Origem do evento|MSSQLSERVER|  
|Componente|SQLEngine|  
|Nome simbólico|DMPIO_BUFFER_OVERFLOW|  
|Texto da mensagem|Um buffer interno ficou cheio.|

# <a name="propertystrvalue-property-sqlserviceadvancedproperty-class"></a>Propriedade PropertyStrValue (classe SqlServiceAdvancedProperty)
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]Obtém o valor de cadeia de caracteres da propriedade avançada.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
object.PropertyStrValue [= value]  
```  
  
## <a name="parts"></a>Partes  
 *objeto*  
 Um [SqlServiceAdvancedProperty classe](../../../relational-databases/wmi-provider-configuration-classes/sqlserviceadvancedproperty-class/sqlserviceadvancedproperty-class.md) objeto que representa uma propriedade avançada.  
  
## <a name="property-valuereturn-value"></a>Valor da propriedade/Valor do retorno  
 Um valor da cadeia de caracteres que especifica o valor da propriedade avançada.

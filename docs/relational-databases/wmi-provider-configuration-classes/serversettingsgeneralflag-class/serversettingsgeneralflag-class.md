# <a name="serversettingsgeneralflag-class"></a>Classe ServerSettingsGeneralFlag
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]O [classe ServerSettingsGeneralFlag](../../../relational-databases/wmi-provider-configuration-classes/serversettingsgeneralflag-class/serversettingsgeneralflag-class.md) representa as configurações gerais de sinalizador em uma instância do [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].  
  
 Essa classe pode ser usada para fazer o seguinte:  
  
-   Exibir o nome do sinalizador.  
  
-   Exibir e modificar o valor de sinalizador.

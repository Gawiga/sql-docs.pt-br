# <a name="cinstance-class"></a>Classe CInstance
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]O [classe CInstance](../../relational-databases/wmi-provider-configuration-classes/cinstance-class.md) representa uma instância de um [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] cliente.  
  
 Esta classe pode ser usada para gerenciar protocolos e bibliotecas de rede associadas com a instância de um cliente do [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)].

# <a name="servernetworkprotocol-class"></a>Classe ServerNetworkProtocol
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]O [classe ServerNetworkProtocol](../../../relational-databases/wmi-provider-configuration-classes/servernetworkprotocol-class/servernetworkprotocol-class.md) representa protocolos de rede instalados em uma instância do [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] que permite a comunicação com clientes em uma rede.  
  
 Com o [ServerNetworkProtocol classe](../../../relational-databases/wmi-provider-configuration-classes/servernetworkprotocol-class/servernetworkprotocol-class.md), você pode:  
  
-   Verificar o nome, a versão, o arquivo .dll, número de propriedades e nome para exibição do protocolo de rede de servidor referenciado.  
  
-   Referenciar as propriedades e endereços IP do protocolo de rede.  
  
-   Habilitar e desabilitar o protocolo de rede de servidor.

# <a name="sqlxml-managed-classes---sqlxmlparameter-object"></a>Classes gerenciadas SQLXML – Objeto SqlXmlParameter
[!INCLUDE[appliesto-ss-asdb-xxxx-xxx-md](../../../includes/appliesto-ss-asdb-xxxx-xxx-md.md)]
  O objeto SqlXmlParameter dá suporte a essas propriedades:  
  
 Nome  
 O nome do parâmetro. Os comandos podem ser parâmetros passados. Chamar o método CreateParameter do objeto SqlXmlCommand cria o objeto de parâmetro.  
  
 Valor  
 O valor do parâmetro. Para obter um exemplo funcional, consulte [executar consultas de SQL &#40;Classes gerenciadas SQLXML&#41;](../../../relational-databases/sqlxml-annotated-xsd-schemas-xpath-queries/net-framework-classes/executing-sql-queries-sqlxml-managed-classes.md).

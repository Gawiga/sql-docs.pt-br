# Proteger uma topologia de replica&#231;&#227;o
  Contas de segurança, logons e senhas necessárias à replicação são fornecidas quando a replicação é configurada e quando as publicações e as assinaturas são criadas. Essas configurações de segurança podem ser alteradas posteriormente usando [!INCLUDE[ssManStudioFull](../../../includes/ssmanstudiofull-md.md)], objetos de gerenciamento de replicação (RMO) e procedimentos armazenados de replicação. Os tópicos a seguir descrevem como modificar as configurações de segurança da replicação.

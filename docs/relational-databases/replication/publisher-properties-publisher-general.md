# Propriedades do Publicador - Publicador, Geral
  O **geral** página do **Propriedades do publicador** caixa de diálogo exibe informações somente leitura no banco de dados do distribuidor e de distribuição que usa o publicador. Para alterar o Distribuidor ou o banco de dados de um Publicador:  
  
1.  Desabilite a publicação no Publicador. Para obter mais informações, consulte [Desabilitar publicação e distribuição](../../relational-databases/replication/disable-publishing-and-distribution.md).  
  
2.  Reconfigure a publicação e a distribuição Para obter mais informações, consulte [Configure Publishing and Distribution](../../relational-databases/replication/configure-publishing-and-distribution.md).

|Nome da coluna|Tipo de dados|Description|Intervalo|  
|-----------------|---------------|-----------------|-----------|  
|**event_name**|**nvarchar(255)**|Nome do evento de diagnóstico específico.||  
|**property_name**|**nvarchar(255)**|Nome de uma propriedade do evento.||

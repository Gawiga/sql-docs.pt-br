# <a name="assemblies---properties"></a>Assemblies - propriedades
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]Use esta página para exibir ou modificar as propriedades para o assembly.  
  
## <a name="options"></a>Opções  
 **Nome do assembly**  
 Exibe o nome do assembly que sempre corresponde ao nome do CLR assembly.  
  
 **Proprietário do assembly**  
 Digite um nome de proprietário ou nome de esquema ou selecione na lista.  
  
 **Conjunto de permissões**  
 Defina o nível de segurança para o assembly. São fornecidos três níveis de segurança: **seguro**, **acesso externo**, e **Unsafe** acesso.  
  
 **Caminho para assembly**  
 Digite o caminho do arquivo assembly.  
  
 **Procurar**  
 Navegue para o assembly que você quer adicionar. Clique em **procurar** se não quiser digitar o caminho para o arquivo de assembly.  
  
## <a name="additional-properties-grid"></a>Grade de propriedades adicionais  
 **Data de criação**  
 Exibe a data em que o assembly foi criado/registrado.  
  
 **Nome forte**  
 Exibe **True** se o assembly foi assinado digitalmente, **False** se ele não foi assinado digitalmente.  
  
 **Versão**  
 Exibe o número de versão do assembly.

# <a name="facet-properties-dialog-box-dependent-conditions-page"></a>Caixa de diálogo Propriedades da Faceta, página Condições Dependentes
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] Use esta caixa de diálogo para exibir uma lista de condições que atualmente referenciam essa faceta do Gerenciamento Baseado em Políticas.  
  
## <a name="options"></a>Opções  
 **Nome**  
 Lista o nome de cada condição que referencia a faceta atual.

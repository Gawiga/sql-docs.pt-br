# <a name="input-and-output-error-during-hard-page-fault"></a>Erro de entrada e saída durante falha de página de hardware
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] Esta regra verifica o log de eventos do sistema para o EventId 51. Este erro é causado por um erro durante uma falha de página de hardware.

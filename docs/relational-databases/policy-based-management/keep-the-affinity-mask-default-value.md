# <a name="keep-the-affinity-mask-default-value"></a>Manter o valor padrão da máscara de afinidade
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] Esta regra verifica se a configuração do [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] para a máscara de afinidade está definida como 0, que é o valor padrão. A opção de máscara de afinidade controla dinamicamente a afinidade de CPU.

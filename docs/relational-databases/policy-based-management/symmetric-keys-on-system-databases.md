# <a name="symmetric-keys-on-system-databases"></a>Chaves simétricas em bancos de dados do sistema
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] Esta regra verifica as chaves simétricas criadas pelo usuário nos bancos de dados mestre, msdb, modelo e tempdb.  
  
## <a name="best-practices-recommendations"></a>Práticas Recomendadas  
 Não crie chaves simétricas nos bancos de dados do sistema.

# <a name="localdberrorinstancebusy"></a>LOCALDB_ERROR_INSTANCE_BUSY
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]
    
## <a name="details"></a>Detalhes  
  
|||  
|-|-|  
|Nome do produto|SQL Server|  
|ID do Evento|274|  
|Origem do evento|Tempo de execução de banco de dados local do SQL Server 12.0|  
|Componente|API do tempo de execução de banco de dados local|  
|Texto da mensagem|A operação solicitada na instância de Banco de Dados Local não pode ser executada porque a instância especificada está em uso no momento. Interrompa a instância e tente novamente.|  
  
## <a name="explanation"></a>Explicação  
 A instância especificada está em execução.  
  
## <a name="user-action"></a>Ação do usuário  
 Interrompa a instância especificada do Tempo de Execução do Banco de Dados Local e tente novamente.

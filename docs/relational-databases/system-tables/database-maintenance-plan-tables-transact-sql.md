# <a name="database-maintenance-plan-tables-transact-sql"></a>Tabelas do plano de manutenção do banco de dados (Transact-SQL)
[!INCLUDE[tsql-appliesto-ss2012-xxxx-xxxx-xxx-md](../../includes/tsql-appliesto-ss2012-xxxx-xxxx-xxx-md.md)]

  Os tópicos desta seção descrevem as tabelas do sistema que armazenam informações utilizadas pelos planos de manutenção do banco de dados. Essas tabelas preservam as informações referentes a instâncias atualizadas a partir de uma versão anterior do [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)].  
  
> [!NOTE]  
>  [!INCLUDE[ssNoteDepNextAvoid](../../includes/ssnotedepnextavoid-md.md)]  
  
## <a name="in-this-section"></a>Nesta seção  
 [sysdbmaintplan_databases](../../relational-databases/system-tables/sysdbmaintplan-databases-transact-sql.md)  
 Contém uma linha para cada banco de dados que tem um plano de manutenção de banco de dados atualizado associado.  
  
 [sysdbmaintplan_history](../../relational-databases/system-tables/sysdbmaintplan-history-transact-sql.md)  
 Contém uma linha para cada ação do plano de manutenção de banco de dados atualizado executada.  
  
 [sysdbmaintplan_jobs](../../relational-databases/system-tables/sysdbmaintplan-jobs-transact-sql.md)  
 Contém uma linha para cada trabalho do plano de manutenção de banco de dados atualizado.  
  
 [sysdbmaintplans](../../relational-databases/system-tables/sysdbmaintplans-transact-sql.md)  
 Contém uma linha para cada plano de manutenção de banco de dados atualizado.

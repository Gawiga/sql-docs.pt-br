# <a name="msdistributor-transact-sql"></a>MSdistributor (Transact-SQL)
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]

  O **MSdistributor** tabela contém as propriedades do distribuidor. Essa tabela é armazenada na **msdb** banco de dados.  
  
|Nome da coluna|Tipo de dados|Description|  
|-----------------|---------------|-----------------|  
|**property**|**sysname**|O nome da propriedade|  
|**value**|**nvarchar(3000)**|O valor da propriedade|

# <a name="developer-guides-for-sql-server"></a>Guias do desenvolvedor do SQL Server
[!INCLUDE[appliesto-ss-asdb-xxxx-xxx-md](../includes/appliesto-ss-asdb-xxxx-xxx-md.md)] Integre quase qualquer recurso do SQL Server por meio de código personalizado usando os modelos e interfaces de programação. Cada área de recurso inclui APIs nativas e gerenciadas usadas para coleta de dados, acesso a dados, administração do servidor ou visualização de dados.  
  
## <a name="sql-server-developer-edition-free"></a>SQL Server Developer Edition (Gratuita!)

- [**Baixar o SQL Server Developer Edition**](https://my.visualstudio.com/Downloads?q=SQL%20Server%20Developer)

- O **SSMS (SQL Server Management Studio)** é um ambiente integrado gratuito para trabalhar com o SQL Server, com versões atualizadas quase mensalmente. Portanto, acesse a [página de download do SSMS](../ssms/download-sql-server-management-studio-ssms.md) para a versão mais recente.

- O **SSDT (SQL Server Data Tools)** também é gratuito e funciona melhor com o Visual Studio. Acesse a [página de download do SSDT](../ssdt/download-sql-server-data-tools-ssdt.md) da versão mais recente, na qual também é possível obter a versão gratuita do **[Visual Studio Community](https://www.visualstudio.com/products/visual-studio-community-vs.aspx)**.

# <a name="developer-guides-for-sql-server"></a>Guias do desenvolvedor do SQL Server
  Integre quase qualquer recurso do SQL Server por meio de código personalizado usando os modelos e interfaces de programação. Cada área de recurso inclui APIs nativas e gerenciadas usadas para coleta de dados, acesso a dados, administração do servidor ou visualização de dados.  
  
## <a name="sql-server-2016-developer-edition-free"></a>SQL Server 2016 Developer Edition (gratuito!)

- [**Download do SQL Server 2016 Developer Edition**](https://my.visualstudio.com/Downloads?q=SQL%20Server%20Developer)

- O **SSMS (SQL Server Management Studio)** é um ambiente integrado gratuito para trabalhar com o SQL Server, com versões atualizadas quase mensalmente. Portanto, acesse a [página de download do SSMS](https://msdn.microsoft.com/library/mt238290.aspx) para a versão mais recente.

- O **SSDT (SQL Server Data Tools)** também é gratuito e funciona melhor com o Visual Studio. Acesse a [página de download do SSDT](https://msdn.microsoft.com/library/mt204009.aspx) para a versão mais recente, você na qual também pode obter a versão gratuita do **[Visual Studio 2015 Community](https://www.microsoft.com/en-us/download/details.aspx?id=48146)**.

# <a name="transact-sql-code-snippets"></a>Trechos de código Transact-SQL
  Um trecho [!INCLUDE[tsql](../../includes/tsql-md.md)].code é um modelo que contém a estrutura básica de um bloco ou uma instrução [!INCLUDE[tsql](../../includes/tsql-md.md)] . Você pode usar os trechos como ponto de partida ao adicionar instruções ao Editor de Consultas do [!INCLUDE[ssDE](../../includes/ssde-md.md)] . Você pode inserir os trechos predefinidos fornecidos com o SQL Server ou criar o seu próprio.  
  
## <a name="transact-sql-code-snippet-tasks"></a>Tarefas de trecho de código Transact-SQL  
  
|Descrição da tarefa|Tópico|  
|----------------------|-----------|  
|Descreve como inserir um trecho existente, como CREATE TABLE ou SELECT.|[Inserir trechos Transact-SQL](../../relational-databases/scripting/insert-transact-sql-snippets.md)|  
|Descreve como incluir um conjunto de instruções [!INCLUDE[tsql](../../includes/tsql-md.md)] em um bloco BEGIN, IF ou WHILE.|[Inserir trechos Transact-SQL com Surround](../../relational-databases/scripting/insert-surround-with-transact-sql-snippets.md)|  
|Descreve como concluir a compilação de um bloco ou uma instrução [!INCLUDE[tsql](../../includes/tsql-md.md)] completa após a inserção de um trecho de código.|[Concluir trechos de código Transact-SQL](../../relational-databases/scripting/complete-transact-sql-snippets.md)|  
|Descreve como compilar seus próprios trechos personalizados e adicioná-los ao conjunto de trechos do SQL Server.|[Adicionar trechos de Transact-SQL](../../relational-databases/scripting/add-transact-sql-snippets.md)|

# Preparar para atualizar executando o Supervisor de Atualiza&#231;&#227;o
  Prepare-se para atualizar seus bancos de dados OLTP e adotar novos recursos no [!INCLUDE[ssSQL15](../../includes/sssql15-md.md)] baixando e executando o Assistente de Migração de Dados da Microsoft.  
  
## <a name="download-and-install-data-migration-assistant"></a>Baixe e instale o Assistente de migração de dados  
 Baixe e instale o Assistente de migração de dados [aqui](http://go.microsoft.com/fwlink/?LinkID=613421). Essa ferramenta não está incluída na mídia de instalação do [!INCLUDE[ssSQL15](../../includes/sssql15-md.md)] .

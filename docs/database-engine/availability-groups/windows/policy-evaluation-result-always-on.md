# Resultado da Avalia&#231;&#227;o de Pol&#237;tica (AlwaysOn)
[!INCLUDE[tsql-appliesto-ss2016-xxxx-xxxx-xxx_md](../../../includes/tsql-appliesto-ss2016-xxxx-xxxx-xxx-md.md)]

  Use a página **Resultado da Avaliação de Política** do Painel do AlwaysOn para ver os problemas de política atuais.  
  
 **Neste tópico:**  
  
-   [Opções de caixas de diálogo](#Options)  
  
-   [Tarefas relacionadas](#RelatedTasks)  
  
##  <a name="Options"></a> Opções de caixas de diálogo  
 **Problema detectado**  
 Exibe uma linha para cada problema detectado. O ícone associado a um problema fornece um indicador visual para indicar a severidade do problema, como segue:  
  
|Ícone|Severity|  
|----------|--------------|  
|![Ícone de IU: erro](../../../database-engine/availability-groups/windows/media/repl-icon-error.png "Ícone de IU: erro")|Crítico|  
|![Ícone de IU: aviso](../../../database-engine/availability-groups/windows/media/repl-icon-warn.png "Ícone de IU: aviso")|Aviso|  
  
 **Descrição**  
 Esta grade exibe uma breve descrição do problema selecionado.  
  
 **Mais informações**  
 Clique neste link para abrir o tópico de ajuda do problema selecionado.

# <a name="analysis-services-samples"></a>Exemplos do Analysis Services
[!INCLUDE[ssas-appliesto-sqlas-all-aas](../includes/ssas-appliesto-sqlas-all-aas.md)]

  Use as informações aqui para localizar bancos de dados de exemplo e projetos para ajudá-lo a conhecer e testar suas soluções do Analysis Services.
  

## <a name="samples-on-github"></a>Exemplos no GitHub

[Repositório do Git para o Analysis Services](https://github.com/Microsoft/Analysis-Services) inclui exemplos de código e projetos da comunidade.

## <a name="sample-solutions-and-databases"></a>Bancos de dados e soluções de exemplo  

Enquanto não é mais oficialmente suporte ou atualizado, Adventure Works continua sendo um dos mais robusto e inclusive exemplo conjuntos de dados para aprender sobre e testar o Analysis Services. Analysis Services de exemplo projetos e bancos de dados, bem como exemplos na documentação, postagens de blog e apresentações usar os dados de exemplo Adventure Works 2014.

[Baixar os projetos de exemplo Adventure Works e bancos de dados no Codeplex](https://msftdbprodsamples.codeplex.com/releases/view/125550).

Uma nova coleção de dados de exemplo Wide World Importers, foi introduzida para SQL Server 2016. Você pode usar dados de exemplo Wide World Importers para saber mais sobre e testar o Analysis Services; No entanto, não há tutoriais, exemplos ou documentação são fornecidos. O conjunto de dados Wide World Importers e o esquema não dão suporte a muitos dos recursos no Analysis Services. Para obter mais informações, consulte [Wide World Importers](https://msdn.microsoft.com/library/mt734199).

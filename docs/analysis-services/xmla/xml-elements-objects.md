# <a name="xml-elements---objects"></a>Elementos XML - objetos
[!INCLUDE[ssas-appliesto-sqlas-aas](../../includes/ssas-appliesto-sqlas-aas.md)]
  O protocolo XML for Analysis (XMLA) usa dois métodos, **Discover** e **Execute**, para oferecer uma maneira padronizada de aplicativos para acessar informações em uma instância do Analysis Services. Uma vez que esses métodos são chamados usando o protocolo SOAP, eles aceitam a entrada e entregam a saída em XML.  
  
## <a name="in-this-section"></a>Nesta seção  
 Os tópicos a seguir descrevem os objetos XMLA implementados pelo Analysis Services.  
  
|Método|Description|  
|------------|-----------------|  
|[Elemento DiscoverResponse &#40;XMLA&#41;](../../analysis-services/xmla/xml-elements-objects-discoverresponse.md)|Contém as informações retornadas por uma instância do Analysis Services em resposta a um [Discover](../../analysis-services/xmla/xml-elements-methods-discover.md) chamada de método.|  
|[Elemento ExecuteResponse &#40;XMLA&#41;](../../analysis-services/xmla/xml-elements-objects-executeresponse.md)|Contém as informações retornadas por uma instância do Analysis Services em resposta a um [Execute](../../analysis-services/xmla/xml-elements-methods-execute.md) chamada de método.|

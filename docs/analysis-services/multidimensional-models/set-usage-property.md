# Definir propriedade de uso
  Você pode definir o uso de um atributo empregando a exibição **Estrutura da Dimensão** do Designer de Dimensão, acessado pelo [!INCLUDE[ssBIDevStudioFull](../../includes/ssbidevstudiofull-md.md)].  
  
 Quando você definir o uso para um atributo, as alterações só entrarão em vigor depois que você processar a dimensão. Para obter mais informações, consulte [Processando um modelo multidimensional &#40;Analysis Services&#41;](../../analysis-services/multidimensional-models/processing-a-multidimensional-model-analysis-services.md).  
  
### Definir o uso para um atributo  
  
1.  No Gerenciador de Soluções, clique com o botão direito do mouse em uma dimensão e clique em **Abrir**.  
  
     A exibição **Estrutura da Dimensão** é aberta por padrão.  
  
2.  Em **Atributos**, clique com o botão direito do mouse no atributo cujo uso você deseja configurar, aponte para **Definir Utilização de Atributo** e, em seguida, clique em uma destas opções:  
  
    -   **Regular**  
  
    -   **Chave**  
  
    -   **Pai**

# Organizar as formas no diagrama de rela&#231;&#227;o do atributo
  Você pode alterar o layout das formas na guia **Relações de Atributo** da exibição Estrutura da Dimensão do Designer de Dimensão no [!INCLUDE[ssBIDevStudioFull](../../includes/ssbidevstudiofull-md.md)].  
  
### Para organizar as formas no diagrama de relação do atributo  
  
1.  No Designer de Dimensão, abra a dimensão que contém os atributos com os quais você deseja criar uma relação de atributo.  
  
2.  Na guia **Relações de Atributo** , no diagrama, clique em uma forma para selecioná-la, capture a imagem pela borda superior e arraste-a para o novo local.  
  
     Você também pode clicar em **Organizar Formas** para organizar automaticamente todas as formas de acordo com o algoritmo de layout que o Designer de Dimensão utiliza.

# <a name="session-events-event-category"></a>categoria Session Events
[!INCLUDE[ssas-appliesto-sqlas](../../includes/ssas-appliesto-sqlas.md)]
  A categoria Session Events tem as classes de evento descritas na tabela a seguir.  
  
|Event Class|ID do evento|Description|  
|-----------------|--------------|-----------------|  
|Existing Connection|41|Coleta todos os eventos de conexão de consulta desde o início do rastreamento.|  
|Existing Session|42|Coleta todos os eventos de sessão desde o início do rastreamento.|  
|Session Initialize|43|Coleta os eventos de inicialização da sessão.|  
  
 Para obter informações sobre as colunas associadas a cada classe de evento Eventos de Sessão, consulte [Colunas de dados de eventos de sessão](../../analysis-services/trace-events/session-events-data-columns.md).

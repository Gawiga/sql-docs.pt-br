# <a name="file-load-and-save-event-category"></a>Categoria de evento File Load and Save
[!INCLUDE[ssas-appliesto-sqlas](../../includes/ssas-appliesto-sqlas.md)]
  A categoria de evento File Load and Save tem as classes de evento descritas na tabela a seguir.  
  
|Event Class|ID do evento|Description|  
|-----------------|--------------|-----------------|  
|File Load Begin|90|File Load Begin|  
|File Load End|91|File Load End|  
|File Save Begin|92|File Save Begin|  
|File Save End|93|File Save End|  
|PageOut Begin|94|PageOut Begin|  
|PageOut End|95|PageOut End|  
|PageIn Begin|96|PageIn Begin|  
|PageIn End|97|PageIn End|  
  
 Para obter informações sobre as colunas associadas a cada classe de evento File Load and Save, consulte [File Load and Save Data Columns](../../analysis-services/trace-events/file-load-and-save-data-columns.md).

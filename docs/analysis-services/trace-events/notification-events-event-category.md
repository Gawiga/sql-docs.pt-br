# <a name="notification-events-event-category"></a>categoria Notification Events
[!INCLUDE[ssas-appliesto-sqlas](../../includes/ssas-appliesto-sqlas.md)]
  A categoria Notification Events tem as classes de evento descritas na tabela a seguir.  
  
|Event Class|ID do evento|Description|  
|-----------------|--------------|-----------------|  
|Notification|39|Coleta todos os eventos de notificação desde o início do rastreamento.|  
|User Defined|40|Coleta todos os eventos definidos pelo usuário desde o início do rastreamento.|  
  
 Para obter informações sobre as colunas associadas a cada classe de evento Eventos de Notificação, consulte [Colunas de dados de eventos de notificação](../../analysis-services/trace-events/notification-events-data-columns.md).

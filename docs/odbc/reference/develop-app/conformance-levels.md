# <a name="conformance-levels"></a>Níveis de conformidade
Drivers ODBC oferecem o acesso do aplicativo para diversas fontes de dados. Cada driver permite que o aplicativo determinar em tempo de execução que recursos ODBC e gramática de SQL que o driver e dá suporte a cada fonte de dados. Isso não é um requisito de aplicativos projetados para trabalhar com um driver único ou um conjunto de drivers, pequeno, conhecido como esses aplicativos simplesmente podem ser gravados em recursos desse driver ou drivers. Para aplicativos de descobrir o driver e recursos de fonte de dados, duas áreas de conformidade estão disponíveis: a interface ODBC e a gramática SQL.

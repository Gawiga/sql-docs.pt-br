# <a name="features-to-watch-for"></a>Recursos para ver
Esta seção descreve uma série de recursos que os desenvolvedores de aplicativos geralmente levam para concedidas. Na verdade, esses recursos variam muito em forma de suporte entre DBMSs; e suporte Falha ao código para eles é pode causar problemas em aplicativos interoperáveis.  
  
 Esta seção não lista todos os recursos que os desenvolvedores de aplicativos precisam considerar. Para obter essas informações, consulte o [SQLGetInfo](../../../odbc/reference/syntax/sqlgetinfo-function.md), [SQLSetStmtAttr](../../../odbc/reference/syntax/sqlsetstmtattr-function.md), e [SQLSetConnectAttr](../../../odbc/reference/syntax/sqlsetconnectattr-function.md) função descrições, [gramática de SQL apêndice c:](../../../odbc/reference/appendixes/appendix-c-sql-grammar.md)e as seções deste manual que abordam cada recurso.

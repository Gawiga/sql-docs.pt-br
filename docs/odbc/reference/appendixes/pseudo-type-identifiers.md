# <a name="pseudo-type-identifiers"></a>Identificadores de tipo pseudo
Para conveniência de programação de aplicativo, o ODBC define um número de identificadores de tipo pseudo. Esses identificadores não correspondem aos tipos de dados reais, na verdade, mas em vez disso, dependendo da situação, resolvem para tipos de dados existentes.  
  
 Esta seção contém os tópicos a seguir.

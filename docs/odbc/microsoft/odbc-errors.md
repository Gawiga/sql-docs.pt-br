# <a name="odbc-errors"></a>Erros de ODBC
Quando ocorre um erro, os Drivers de banco de dados de área de trabalho do Microsoft ODBC retornar o número de erro nativo, o SQLSTATE (um código de erro ODBC) e uma mensagem de erro.

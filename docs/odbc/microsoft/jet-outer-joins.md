# <a name="jet-outer-joins"></a>Jet: junções externas
Uma instrução SELECT pode conter uma lista de cláusulas de junção externa. Há suporte para junções externas aninhadas. Para obter mais informações sobre junções externas, consulte o *guia do programador do Microsoft Jet banco de dados Engine*.

# <a name="sqlbindparameter-visual-foxpro-odbc-driver"></a>SQLBindParameter (Driver ODBC do Visual FoxPro)
> [!NOTE]  
>  Este tópico contém informações específicas de Driver ODBC do Visual FoxPro. Para obter informações gerais sobre essa função, consulte o tópico apropriado sob [referência da API ODBC](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Suporte: completo  
  
 Conformidade com a API ODBC: 1 de nível  
  
 Associa um buffer com um marcador de parâmetro em uma instrução SQL. O Driver de ODBC do Visual FoxPro dá suporte a parâmetros de entrada conforme especificado pelo *fParamType* argumento.  
  
 Para obter mais informações, consulte [SQLBindParameter](../../odbc/reference/syntax/sqlbindparameter-function.md) na *referência do programador de ODBC*.

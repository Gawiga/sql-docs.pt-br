# <a name="creating-and-dropping-views-and-procedures-using-extendedansisql"></a>Criar e remover exibições e procedimentos usando o ExtendedAnsiSQL
Quando o sinalizador ExtendedAnsiSQL é ativado, Jet 4.0 oferece suporte a capacidade de criar e descartar exibições e procedimentos por meio do SQL. Essas exibições e procedimentos não será visíveis por meio do acesso.

# <a name="convert-function-limitations"></a>Limitações da função CONVERT
Tipo de resultado de falhas de conversão da coluna em questão que está sendo definida como NULL.  
  
 Tipo de dados nem o data nem o carimbo de hora pode ser convertido em outro data tipo (ou em si) pela função CONVERT.

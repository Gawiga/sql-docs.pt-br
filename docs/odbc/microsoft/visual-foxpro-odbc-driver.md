# <a name="visual-foxpro-odbc-driver"></a>Driver ODBC do Visual FoxPro
Microsoft Visual FoxPro é um ambiente poderoso e orientada a objeto para o desenvolvimento de construção e o aplicativo de banco de dados. O Driver ODBC do Microsoft Visual FoxPro permite que os aplicativos abrir a consulta e atualizar dados do Visual FoxPro e versões anteriores do FoxPro através da interface do banco de dados ODBC (conectividade aberta).  
  
 Por exemplo, com o Driver ODBC do Microsoft Visual FoxPro você pode fazer o seguinte:  
  
-   Use o Microsoft Query para consultar e atualizar dados do Visual FoxPro planilhas do Microsoft Excel.  
  
-   Crie cartas de mala direta usando dados do Visual FoxPro com o Microsoft Word.  
  
-   Consultar e atualizar tabelas do Microsoft Access e modos de exibição do Visual FoxPro.  
  
-   Uso do Visual FoxPro como o repositório de dados para aplicativos do Microsoft Visual Basic, Microsoft Visual C++ e C.

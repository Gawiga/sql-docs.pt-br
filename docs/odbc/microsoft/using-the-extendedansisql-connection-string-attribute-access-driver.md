# <a name="using-the-extendedansisql-connection-string-attribute-access-driver"></a>Usando o atributo de cadeia de caracteres de Conexão ExtendedAnsiSQL (Driver de acesso)
Jet 4.0 tem vários recursos novos que não são habilitados por padrão em Drivers de banco de dados do ODBC Desktop versão 4.0. Para aproveitar esses novos recursos, um aplicativo precisa definir um novo atributo de conexão, adicionar um novo atributo para a cadeia de caracteres de conexão ou defina o atributo em um DSN. O atributo é chamado ExtendedAnsiSQL e diz respeito apenas ao Microsoft ODBC Desktop Database Drivers 4.0 e posterior.

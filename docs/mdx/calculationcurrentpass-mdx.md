# <a name="calculationcurrentpass-mdx"></a>CalculationCurrentPass (MDX)


  Retorna a análise de cálculo atual de um cubo para o contexto de consulta especificado.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
CalculationCurrentPass()  
```  
  
## <a name="remarks"></a>Remarks  
 O **CalculationCurrentPass** função retorna o índice baseado em zero da fase de cálculo para o contexto de consulta atual. Com a resolução de recursão automática, esta função tem pouco uso prático.

# <a name="enable-or-disable-profiling-notifications-in-dqs"></a>Habilitar ou desabilitar notificações de criação de perfil no DQS
  Este tópico descreve como habilitar ou desabilitar notificações de criação de perfil no [!INCLUDE[ssDQSnoversion](../includes/ssdqsnoversion-md.md)] (DQS). Por padrão, as notificações de criação de perfil estão habilitadas no DQS. As notificações de criação de perfil informam fatos importantes sobre a fonte de dados e a efetividade da atividade atual executada nos dados. Para obter mais informações, consulte [criação de perfil de dados e notificações no DQS](../../2014/data-quality-services/data-profiling-and-notifications-in-dqs.md).  
  
##  <a name="BeforeYouBegin"></a> Antes de começar  
  
###  <a name="Security"></a> Segurança  
  
####  <a name="Permissions"></a> Permissões  
 Você deve ter a função dqs_administrator no banco de dados DQS_MAIN para habilitar as notificações.  
  
##  <a name="Enable"></a> Habilitar ou desabilitar notificações de criação de perfil  
  
1.  [!INCLUDE[ssDQSInitialStep](../includes/ssdqsinitialstep-md.md)] [Executar o aplicativo Data Quality Client](../../2014/data-quality-services/run-the-data-quality-client-application.md).  
  
2.  Na tela inicial do [!INCLUDE[ssDQSClient](../includes/ssdqsclient-md.md)] , clique em **Configuração**.  
  
3.  Em seguida, clique na guia **Configurações Gerais** .  
  
4.  Marque ou desmarque a caixa de seleção **Habilitar Notificações** para desabilitar ou habilitar as notificações de criação de perfil para várias atividades no DQS.  
  
5.  Clique em **Fechar**.

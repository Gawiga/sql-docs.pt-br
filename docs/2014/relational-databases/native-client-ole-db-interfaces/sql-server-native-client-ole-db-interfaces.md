# <a name="sql-server-native-client-ole-db-reference"></a>Referência do SQL Server Native Client (OLE DB)
  O [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] provedor do OLE DB Native Client expõe as interfaces para consumidores que desejam acesso a dados em um ou mais computadores que executam o [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]. O provedor OLE DB do [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Native Client é compatível com o OLE DB versão 2.0.  
  
 Isso [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] referência de programação do provedor de OLE DB Native Client não documenta todas as interfaces do OLE DB e métodos, somente as interfaces e métodos que exibem um comportamento específico do provedor ao usar o [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Native Client OLE DB provedor.  
  
 Para obter uma descrição completa das interfaces do OLE DB de núcleo e métodos, consulte o [referência do programador de DB OLE](http://go.microsoft.com/fwlink/?LinkId=45232).

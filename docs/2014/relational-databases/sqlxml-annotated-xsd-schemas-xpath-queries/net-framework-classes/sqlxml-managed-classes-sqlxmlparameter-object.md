# <a name="sqlxmlparameter-object-sqlxml-managed-classes"></a>Objeto SqlXmlParameter (classes gerenciadas do SQLXML)
  O objeto SqlXmlParameter dá suporte a essas propriedades:  
  
 Nome  
 O nome do parâmetro. Os comandos podem ser parâmetros passados. Chamar o método CreateParameter do objeto SqlXmlCommand cria o objeto de parâmetro.  
  
 Valor  
 O valor do parâmetro. Para obter um exemplo de funcionamento, consulte [executar consultas de SQL &#40;Classes gerenciadas SQLXML&#41;](sqlxml-4-0-net-framework-support-managed-classes.md).

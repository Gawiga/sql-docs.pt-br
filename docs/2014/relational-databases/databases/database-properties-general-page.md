# <a name="database-properties-general-page"></a>Propriedades de Banco de Dados (página Geral)
  Use essa página para exibir ou modificar propriedades do banco de dados selecionado.  
  
## <a name="options"></a>Opções  
 **Último backup de banco de dados**  
 Exibe a data do último backup do banco de dados.  
  
 **Último backup de log do banco de dados**  
 Exibe a data do último backup do log de transações do banco de dados.  
  
 **Nome**  
 Exibe o nome do banco de dados.  
  
 **Status**  
 Exibe o estado do banco de dados. Para obter mais informações, consulte [Database States](database-states.md).  
  
 **Proprietário**  
 Exibe o nome do proprietário do banco de dados. O proprietário pode ser alterado na página **Arquivos** .  
  
 **Data da Criação**  
 Exibe data e hora de criação do banco de dados.  
  
 **Tamanho**  
 Exibe o tamanho do banco de dados em megabytes.  
  
 **Espaço Disponível**  
 Exibe o espaço disponível no banco de dados em megabytes.  
  
 **Número de Usuários**  
 Exibe o número de usuários conectados ao banco de dados.  
  
 **Nome da Ordenação**  
 Exibe a ordenação usada para o banco de dados. A ordenação pode ser alterada na página **Opções**.

# <a name="localdberrorcannotaccessinstancefolder"></a>LOCALDB_ERROR_CANNOT_ACCESS_INSTANCE_FOLDER
    
## <a name="details"></a>Detalhes  
  
|||  
|-|-|  
|Nome do produto|SQL Server|  
|ID do evento|261|  
|Origem do evento|Tempo de execução de banco de dados local do SQL Server 12.0|  
|Componente|API do tempo de execução de banco de dados local|  
|Texto da mensagem|Não é possível acessar a pasta da instância do banco de dados Local: %%LOCALAPPDATA%%\Microsoft\Microsoft DB\Instances Local do SQL Server\\< nome da instância\>.|  
  
## <a name="explanation"></a>Explicação  
 Não é possível acessar uma pasta de instância.

## <a name="details"></a>Detalhes

|-|-|  
|Nome do produto|[!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]|

|Origem do evento|MSSQLSERVER|

|Nome simbólico|HK_ENCRYPTION_ON|

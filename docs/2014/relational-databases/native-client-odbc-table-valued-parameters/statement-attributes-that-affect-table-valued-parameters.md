|Nome do atributo|Tipo|Description|  
|--------------------|----------|-----------------|  
|SQL_SOPT_SS_PARAM_FOCUS|SQLUINTEGER|Para obter mais informações sobre SQL_SS_PARAM_FOCUS, consulte [SQLSetStmtAttr](../native-client-odbc-api/sqlsetstmtattr.md).|

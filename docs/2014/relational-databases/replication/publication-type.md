# <a name="publication-type"></a>Tipo de Publicação
  A replicação fornece os tipos seguintes de publicações:  
  
-   Publicação de instantâneo  
  
-   Publicação transacional  
  
-   Publicação de mesclagem  
  
 O tipo ou tipos de replicação que você escolhe para seu aplicativo dependem do ambiente físico da replicação, do tipo e da quantidade de dados a serem replicados e se os dados serão ou não atualizados no Assinante. O ambiente físico inclui o número e local dos computadores envolvidos na replicação e se esses computadores são clientes (estações de trabalho, laptops ou dispositivos portáteis) ou servidores. Para obter mais informações, consulte [Types of Replication](types-of-replication.md) (Tipos de replicação).  
  
## <a name="options"></a>Opções  
 **Publication type**  
 Selecione o tipo de replicação apropriado para essa publicação.

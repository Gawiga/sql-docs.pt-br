# <a name="context-property-securitycertificate-class"></a>Propriedade de contexto (classe SecurityCertificate)
  Obtém o contexto do certificado de segurança.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
object  
.Context [= value]  
```  
  
## <a name="parts"></a>Partes  
 *object*  
 Um objeto da [classe SecurityCertificate](securitycertificate-class.md) que representa um certificado de segurança.  
  
## <a name="property-valuereturn-value"></a>Valor da propriedade/Valor do retorno  
 Um valor de matriz `sint32` que especifica o contexto do certificado de segurança.

# <a name="protocolname-property-servernetworkprotocolproperty-class"></a>Propriedade ProtocolName (classe ServerNetworkProtocolProperty)
  Obtém o nome do protocolo que possui a propriedade referenciada.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
object  
.PropertyName [= value]  
```  
  
## <a name="parts"></a>Partes  
 *object*  
 A [classe ServerNetworkProtocolProperty](servernetworkprotocolproperty-class.md) que representa um atributo do protocolo de rede na instância do [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].  
  
## <a name="property-valuereturn-value"></a>Valor da propriedade/Valor do retorno  
 Um valor da cadeia de caracteres que especifica o nome do protocolo que possui a propriedade.

# <a name="advancedproperties-property-sqlservice-class"></a>Propriedade AdvancedProperties (classe SqlService)
  Obtém uma matriz de referências de objeto que contém as propriedades avançadas para o objeto `SqlService`.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
object  
.AdvancedProperties [= value]  
```  
  
## <a name="parts"></a>Partes  
 *object*  
 Um objeto da [classe SqlService](sqlservice-class.md) que representa o serviço.  
  
## <a name="property-valuereturn-value"></a>Valor da propriedade/Valor do retorno  
 Uma matriz de [classe SqlServiceAdvancedProperty](../sqlserviceadvancedproperty-class/sqlserviceadvancedproperty-class.md) objetos que contêm as propriedades avançadas para o `SqlService` objeto.

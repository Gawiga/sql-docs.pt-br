# <a name="servernetworkprotocol-class"></a>Classe ServerNetworkProtocol
  O [classe ServerNetworkProtocol] servernetworkprotocol-class.md) representa protocolos de rede instalados em uma instância do [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] que permite a comunicação com clientes em uma rede.  
  
 Com servernetworkprotocol a [classe ServerNetworkProtocol]-class.md), você pode:  
  
-   Verificar o nome, a versão, o arquivo .dll, número de propriedades e nome para exibição do protocolo de rede de servidor referenciado.  
  
-   Referenciar as propriedades e endereços IP do protocolo de rede.  
  
-   Habilitar e desabilitar o protocolo de rede de servidor.

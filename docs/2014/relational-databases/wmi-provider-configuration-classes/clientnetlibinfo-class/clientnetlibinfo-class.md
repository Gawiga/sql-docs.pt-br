# <a name="clientnetlibinfo-class"></a>Classe ClientNetLibInfo
  A [classe ClientNetLibInfo](clientnetlibinfo-class.md) representa informações sobre a biblioteca de rede do cliente.  
  
 O arquivo de biblioteca de rede de cliente deixa a instância de cliente se comunicar em uma rede com uma instância do [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] usando um protocolo de rede específico.

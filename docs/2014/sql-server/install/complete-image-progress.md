# <a name="complete-image-progress"></a>Andamento da Imagem Completa
  Use a página **Andamento da Imagem Completa** do Assistente de Instalação do [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] para monitorar o status da instalação da conclusão da imagem do [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] . Após a conclusão desta etapa, esta página listará o status para cada recurso do [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] configurado e concluído.

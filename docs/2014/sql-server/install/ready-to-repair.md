# <a name="ready-to-repair"></a>Pronto para Reparar
  A página Pronto para Reparar fornece um resumo do processo de reparo através da Instalação do [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)].  
  
## <a name="options"></a>Opções  
 Você tem as seguintes opções nessa página:  
  
-   Para continuar a Instalação, clique em **Reparar**.  
  
-   Para alterar suas configurações de instalação, clique em **Voltar**.  
  
-   Para encerrar a Instalação, clique em **Cancelar**

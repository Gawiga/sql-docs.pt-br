# <a name="task-5-adding-data-flow-task"></a>Tarefa 5: Adicionando a tarefa de fluxo de dados
  Nesta tarefa, você adicionará uma Tarefa de Fluxo de Dados ao fluxo de controle do pacote SSIS.  
  
1.  Arrastar e soltar **a tarefa de fluxo de dados** de **caixa de ferramentas do SSIS** para o **fluxo de controle** guia no Designer SSIS. Se você não vir o **caixa de ferramentas do SSIS**, clique em qualquer lugar do **fluxo de controle** , clique em **SSIS** na barra de menus, clique em **caixa de ferramentas do SSIS**.  
  
     ![Fluxo de dados de arrastar e soltar tarefa da caixa de ferramentas do SSIS](../../2014/tutorials/media/et-addingdataflowtask.jpg "tarefa da caixa de ferramentas do SSIS de fluxo de dados de arrastar e soltar")  
  
2.  Com o botão direito do **a tarefa de fluxo de dados** no **fluxo de controle** guia e clique em **Renomear**.  
  
3.  Tipo **receber, limpar, correspondência e corrigir dados do fornecedor** e pressione **ENTER**.  
  
4.  Clique duas vezes o **a tarefa de fluxo de dados** para alternar para o **de fluxo de dados** guia.

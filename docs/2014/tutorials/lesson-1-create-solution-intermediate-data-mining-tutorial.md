# <a name="lesson-1-creating-the-intermediate-data-mining-solution-intermediate-data-mining-tutorial"></a>Lição 1: Criando a solução intermediária de data mining (Tutorial de data mining intermediário)
  O tutorial de mineração de dados básica, você criou uma [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] projeto que contém uma solução de mineração de dados simples com base na nova [!INCLUDE[ssSampleDBDWobject](../includes/sssampledbdwobject-md.md)] banco de dados.  
  
 Para este tutorial, você criará um separado [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] projeto usando [!INCLUDE[ssBIDevStudioFull](../includes/ssbidevstudiofull-md.md)]. Você aprenderá a criar uma [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] fonte de dados que usa [!INCLUDE[ssSampleDBDWobject](../includes/sssampledbdwobject-md.md)]e adicionar várias exibições da fonte de dados novas a essa fonte de dados, para dar suporte a cenários e tipos de modelos.  
  
 Esta lição consiste na seguinte tarefa:  
  
-   [Criando uma solução e a fonte de dados &#40;Tutorial de mineração de dados intermediário&#41;](../../2014/tutorials/creating-a-solution-and-data-source-intermediate-data-mining-tutorial.md)

# <a name="go-to-dialog-box-ssas"></a>Caixa de diálogo Ir para (SSAS)
  A caixa de diálogo **Ir para** o habilita a saltar para uma coluna específica em uma tabela.  
  
## <a name="uielement-list"></a>Lista de elementos de interface do usuário  
  
|||  
|-|-|  
|Termo|Definição|  
|**Ir para**|Selecione uma coluna na lista **Ir para** e clique em **OK**.|

# <a name="queries-events-category"></a>Categoria Queries Events
  A categoria Queries Events tem as classes de evento descritas na tabela a seguir.  
  
|Event Class|ID do evento|Description|  
|-----------------|--------------|-----------------|  
|Query Begin|9|Coleta todos os eventos de início de consulta desde o início do rastreamento.|  
|Query End|10|Coleta todos os eventos de término de consulta desde o começo do rastreamento.|  
  
 Para obter informações sobre as colunas associadas a cada classe Query Events, consulte [Queries Events Data Columns](queries-events-data-columns.md).

# <a name="discover-events-event-category"></a>categoria Discover Events
  A categoria Discover Events tem as classes de evento descritas na tabela a seguir.  
  
|Event Class|ID do evento|Description|  
|-----------------|--------------|-----------------|  
|Discover Begin|36|Coleta todos os eventos de início de identificação desde o começo do rastreamento.|  
|Discover End|38|Coleta todos os eventos de término de identificação desde o começo do rastreamento.|  
  
 Para obter informações sobre as colunas associadas a cada classe de evento Discover Events, consulte [Colunas de dados Discover Events](discover-events-data-columns.md).

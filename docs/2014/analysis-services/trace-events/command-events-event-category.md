# <a name="command-events-event-category"></a>categoria Command Events
  As classes de evento da categoria de evento Eventos de Comando estão descritas na tabela a seguir.  
  
|Event Class|ID do evento|Description|  
|-----------------|--------------|-----------------|  
|Command Begin|15|Coleta todos os eventos de início de comando desde o começo do rastreamento.|  
|Command End|16|Coleta todos os eventos de término de comando desde o começo do rastreamento.|  
  
 Para obter informações sobre as colunas associadas a cada classe de evento Eventos de Comando, consulte [Command Events Data Columns](command-events-data-columns.md).

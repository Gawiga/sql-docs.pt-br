# <a name="editionid-element"></a>Elemento EditionId
  Contém o código de edição numérica do identificador do pai [Server Element (ASSL)](../../scripting/objects/server-element-assl.md) elemento.  
  
## <a name="syntax"></a>Sintaxe  
  
```xml  
  
<Server>  
   ...  
   <EditionId>...</EditionId>  
   ...  
</Server>  
```  
  
## <a name="element-characteristics"></a>Características do elemento  
  
|Característica|Description|  
|--------------------|-----------------|  
|Comprimento e tipo de dados|Longo|  
|Valor padrão|None|  
|Cardinalidade|1-1: elemento obrigatório que ocorre apenas uma única vez.|  
  
## <a name="element-relationships"></a>Relações do elemento  
  
|Relação|Elemento|  
|------------------|-------------|  
|Elementos pai|[Elemento Server (ASSL)](../../scripting/objects/server-element-assl.md)|  
|Elementos filho|None|

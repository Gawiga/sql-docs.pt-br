# <a name="visibility-element-assl"></a>Elemento Visibility (ASSL)
  Define a visibilidade de um [anotação](../objects/annotation-element-assl.md) elemento.  
  
## <a name="syntax"></a>Sintaxe  
  
```xml  
  
<Annotation>  
   ...  
   <Visibility>...</Visibility>  
   ...  
</Annotation>  
```  
  
## <a name="element-characteristics"></a>Características do elemento  
  
|Característica|Description|  
|--------------------|-----------------|  
|Comprimento e tipo de dados|Cadeia de caracteres (enumeração)|  
|Valor padrão|*Nenhuma*|  
|Cardinalidade|0-1: elemento opcional que pode ocorrer apenas uma única vez.|  
  
## <a name="element-relationships"></a>Relações do elemento  
  
|Relação|Elemento|  
|------------------|-------------|  
|Elemento pai|[Anotação](../objects/annotation-element-assl.md)|  
|Elementos filho|None|  
  
## <a name="remarks"></a>Comentários  
 O valor desse elemento é limitado a uma das cadeias de caracteres listadas na tabela a seguir.  
  
|Valor|Description|  
|-----------|-----------------|  
|*SchemaRowset*|A anotação é visível no conjunto de linhas do esquema.|  
|*Nenhuma*|A anotação não é visível.|  
  
 A enumeração que corresponde aos valores permitidos para `Visibility` no objeto Analysis Management Objects (AMO) o modelo é <xref:Microsoft.AnalysisServices.Annotation>.

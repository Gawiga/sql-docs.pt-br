# <a name="proxy-account-properties-and-new-proxy-account-principals-tab"></a>Propriedades da conta proxy e a nova conta Proxy (guia principais)
  Use essa página para exibir ou alterar os principais que podem usar uma conta proxy do [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Agent em etapas de trabalho.  
  
## <a name="options"></a>Opções  
 **Principais da conta proxy**  
 Lista os principais que podem usar essa conta proxy.  
  
 **Adicionar**  
 Adiciona um principal à lista.  
  
 **Remover**  
 Remove o principal selecionado da lista.

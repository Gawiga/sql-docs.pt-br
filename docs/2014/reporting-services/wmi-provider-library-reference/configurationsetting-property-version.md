# <a name="version-property--wmi-msreportserverconfigurationsetting"></a>Propriedade Version (WMI MSReportServer_ConfigurationSetting)
  Retorna a versão do servidor de relatório no formato Major.Minor.Build.Revision. Somente leitura.  
  
## <a name="syntax"></a>Sintaxe  
  
```vb  
Public Dim Version As String  
```  
  
```csharp  
public string Version;  
```  
  
## <a name="property-value"></a>Valor da propriedade  
 Um `string` que contém a versão do servidor de relatório.

# <a name="installing-ssma-components-on-sql-server-db2tosql"></a>Instalar os componentes do SSMA no SQL Server (DB2ToSQL)
Nesta versão do SSMA não é necessário para uma instalação separada em [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)] do pacote de extensão do SSMA, que dá suporte à migração de dados e provedores do DB2 para permitir a conectividade de servidor para servidor.  
  
## <a name="ssma-for-db2-extension-pack"></a>SSMA para o pacote de extensão do DB2  
O pacote de extensão do SSMA adiciona um esquema de banco de dados na instância especificada do [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)]. O **sysdb** esquema contém as tabelas e procedimentos armazenados que são necessárias para migrar dados e as funções definidas pelo usuário que emulam funções de sistema do DB2.

# <a name="what39s-new-in-ssma--for-db2-db2tosql"></a>O que &#39; s do SSMA para DB2 (DB2ToSQL)
Este tópico lista SSMA para DB2 alterações em cada versão.  

## <a name="may-2016"></a>Maio de 2016  
A versão de maio de 2016 do SSMA for DB2 contém as seguintes alterações:  

1.  Adicionado suporte para o SQL Server 2016
2.  Conversão adicional de tabelas na memória e regulares do DB2 aos recursos de memória e hekaton do SQL Server
3.  Conversão adicional DB2 de controles de acesso a objetos de política do SQL Server (segurança em nível de linha para DB2)
4.  Conversão adicional de tabelas de versão de sistema do DB2 para tabelas temporais do SQL Server
5.  Analisador do DB2 e o resolvedor aprimorado
6.  Removida a seleção de instalador para .net 2.0
7.  Removido dll desnecessários do instalador do Db2
8.  Fixa "salvar o projeto" e "Abrir projeto" comandos do Console SSMA
9.  Comando fixa "securepassword" para o Console do SSMA
10. Fixa a contagem de objetos para o carregamento inicial
11. Correção do bug nas configurações globais
  
## <a name="march-2016"></a>Março de 2016  
A versão de visualização de março de 2016 do SSMA for DB2 contém as seguintes alterações:  
  
1.  Suporte à migração para o SQL Server 2016  
  
## <a name="january-2016"></a>Janeiro de 2016  
A versão de manutenção de janeiro de 2016 do SSMA for DB2 contém as seguintes alterações:  
  
1.  Adicionado suporte para um número de funções padrão  
  
2.  Erros de analisador DB2 fixa  
  
3.  ZOS de v9 fixa DB2 suporte (RFC 5690920)  
  
4.  Erros de identificador não resolvidos de DB2 fixo durante a conversão  
  
5.  Item de Menu de Log de exibição adicionado para o SSMA (RFC 5706203)  
  
6.  Telemetria adicionada  
  
## <a name="november-2014"></a>Novembro de 2014  
A versão de novembro de 2014 do SSMA for DB2 é o primeiro desse tipo.

# <a name="refresh-from-database-accesstosql"></a>Atualização do banco de dados (AccessToSQL)
O **de atualização do banco de dados** caixa de diálogo permite que você selecione quais objetos para atualização do banco de dados do Access. Linhas na caixa de diálogo estão codificados por cores com base no estado de metadados:  
  
-   Se os metadados do objeto foi alterado localmente e o banco de dados do Access, a linha azul.  
  
-   Se os metadados do objeto foi alterado no banco de dados do Access, mas não em SSMA, a linha é amarela.  
  
-   Se os metadados do objeto foi alterado localmente, mas não no banco de dados de acesso, a linha é verde.  
  
-   Se o objeto for novo no banco de dados do Access, a linha é rosa.  
  
Você pode especificar as configurações padrão do objeto atualização o **configurações de projeto** caixa de diálogo. Para obter mais informações, consulte [configurações de projeto &#40;carregar objetos&#41; &#40;AccessToSQL&#41;](../../ssma/access/project-settings-loading-objects-accesstosql.md)  
  
Para acessar o **de atualização do banco de dados** caixa de diálogo caixa, clique em qualquer **banco de dados** nó no Gerenciador de metadados de acesso e clique em **de atualização do banco de dados**.

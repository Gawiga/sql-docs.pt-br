# <a name="boolean-constructor-functions---false-xquery"></a>Funções do construtor Boolianas - false (XQuery)
[!INCLUDE[tsql-appliesto-ss2012-xxxx-xxxx-xxx-md](../includes/tsql-appliesto-ss2012-xxxx-xxxx-xxx-md.md)]

  Retorna o valor xs:boolean False. Isso é equivalente a `xs:boolean("0")`.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
fn:false() as xs:boolean  
```  
  
## <a name="examples"></a>Exemplos  
 Este tópico fornece exemplos de XQuery em instâncias XML que são armazenados em várias **xml** colunas de tipo de banco de dados AdventureWorks.

# <a name="validate-expressions-xquery"></a>Expressões de validade (XQuery)
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]

  Nessa implementação, o **validar** não há suporte para a expressão. Os resultados das expressões de construção XQuery são sempre não digitados. Se o resultado de uma expressão XQuery deve ser digitado, use a expressão SQL CAST para converter o resultado em uma **xml** tipo de dados com a coleção de esquemas preferida.
